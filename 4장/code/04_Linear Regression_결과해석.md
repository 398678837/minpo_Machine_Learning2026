## 7. RMSE (Root Mean Squared Error)

```
5684.927776334484
```

### 해석

- **RMSE = $5,684.93**
- 이는 "**평균적으로 예측이 실제값에서 약 $5,685 정도 벗어난다**"는 의미이다
- 평균 보험료가 $13,270이므로, RMSE/평균 = 5,685/13,270 = **약 42.8%** → 상당히 큰 오차
- RMSE에 대한 절대적인 좋다/나쁘다 기준은 없으며, **다른 모델과의 상대 비교**에 사용된다
- 향후 Ridge, Lasso, Random Forest, XGBoost 등을 적용하여 이 RMSE를 낮추는 것이 목표가 된다

### RMSE 계산 과정

$$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$

1. 각 데이터에 대해 (실제값 - 예측값)의 차이를 구한다
2. 차이를 제곱한다 (부호 제거 + 큰 오차에 패널티)
3. 제곱한 값들의 평균을 구한다 (MSE)
4. 평균에 루트를 씌운다 (원래 데이터 스케일로 복원)

## 8. R² (결정 계수)

```
0.7368220127747351
```

### 해석

- **R² = 0.7368 (약 73.7%)**
- 이는 "**독립변수(age, sex, bmi, children, smoker) 5개가 종속변수(charges)의 변동 중 약 73.7%를 설명한다**"는 의미이다

$$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$$

- **SST** (Sum of Squares Total): 종속변수의 총 변동량 (평균값 대비)
- **SSR** (Sum of Squares Regression): 모델이 설명하는 변동량
- **SSE** (Sum of Squares Error): 모델이 설명하지 못하는 변동량 (오차)

| R² 범위 | 해석 |
|--------|------|
| 0.9 이상 | 매우 우수한 모델 |
| 0.7 ~ 0.9 | 양호한 모델 |
| 0.5 ~ 0.7 | 보통 |
| 0.5 미만 | 설명력 부족 |

- 우리 모델의 R² = 0.74는 **"양호" 수준**에 해당한다
- **남은 26.3%**는 현재 모델에 포함되지 않은 요인들로 설명된다:
  - 기저 질환 유무, 거주 지역, 직업, 운동 습관 등
  - 또는 변수 간의 비선형 관계 (예: 흡연자의 BMI 효과가 비흡연자와 다를 수 있음)

### 핵심 인사이트

3. **BMI와 나이가 비슷한 영향**: 각각 298, 265로 유사한 크기의 영향을 미친다.
4. **자녀 수도 유의미한 영향**: 자녀 1명당 약 $469 증가 → 부양가족에 따른 보험료 산정이 반영됨.

## 10. 절편 (Intercept)

```
-11576.999976112374
```

### 해석

- **절편(intercept) = -$11,577**
- 이는 수식에서 **모든 독립변수가 0일 때의 charges 값**을 나타낸다
- 실제로 "0세, 여성, BMI 0, 자녀 0, 비흡연"인 사람은 존재하지 않으므로, 절편 자체에 현실적 의미를 부여하기보다는 **수학적 보정값**으로 이해하는 것이 적절하다
- 음수인 이유: 다른 계수들의 합이 양수로 크기 때문에 절편이 음수가 되어야 전체 예측값이 합리적인 범위에 들어온다

## 11. 최종 예측 수식

$$charges = 264.80 \times age + 17.34 \times sex + 297.51 \times bmi + 469.34 \times children + 23469.28 \times smoker - 11577.00$$

### 예측 예시

**30세, 남성, BMI 25, 자녀 1명, 비흡연자**의 예측 보험료:

$$charges = 264.80 \times 30 + 17.34 \times 1 + 297.51 \times 25 + 469.34 \times 1 + 23469.28 \times 0 - 11577.00$$
$$= 7944.00 + 17.34 + 7437.75 + 469.34 + 0 - 11577.00$$
$$= \$4,291.43$$

**같은 조건의 흡연자**라면:

$$charges = 4291.43 + 23469.28 = \$27,760.71$$

→ 흡연 여부 하나로 보험료가 **약 6.5배** 차이가 난다!

## 12. 종합 평가

### 모델 성능 요약

| 지표 | 값 | 평가 |
|------|------|------|

### 모델의 강점
- 해석이 매우 직관적: 각 변수가 보험료에 미치는 영향을 정량적으로 파악 가능
- 훈련 속도가 매우 빠름
- 흡연이 보험료의 핵심 결정 요인임을 명확히 포착

### 모델의 한계
- 흡연자 내에서의 세밀한 예측이 부족 (비선형 관계 미포착)
- charges가 우편향(right-skewed) 분포이므로 고액 보험료 구간에서 예측 오차가 커짐
- 교호작용(interaction effect)을 고려하지 못함 (예: 흡연자의 BMI 효과가 더 클 수 있음)

### 개선 방향
1. **로그 변환**: charges에 log를 취하여 분포를 정규화
2. **교호작용 추가**: smoker × bmi 등의 상호작용 항 추가
3. **비선형 모델**: 의사결정나무, 랜덤 포레스트, XGBoost 등으로 비선형 관계 포착
4. **정규화 기법**: Ridge, Lasso, Elastic Net으로 일반화 성능 향상
