## 7. RMSE (Root Mean Squared Error)

```
5684.927776334484
```

### 解析

- **RMSE = $5,684.93**
- 这意味着"**预测值与实际值平均偏差约为 $5,685**"
- 平均保险费为 $13,270，因此 RMSE/平均值 = 5,685/13,270 = **约42.8%** → 误差相当大
- RMSE没有绝对的好/坏标准，主要用于**与其他模型的相对比较**
- 后续应用Ridge、Lasso、Random Forest、XGBoost等方法来降低此RMSE将成为目标

### RMSE 计算过程

$$RMSE = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2}$$

1. 对每个数据计算（实际值 - 预测值）的差
2. 对差值进行平方（消除符号 + 对大误差施加惩罚）
3. 计算平方值的平均（MSE）
4. 对平均值取平方根（恢复到原始数据尺度）

## 8. R²（决定系数）

```
0.7368220127747351
```

### 解析

- **R² = 0.7368（约73.7%）**
- 这意味着"**5个自变量（age、sex、bmi、children、smoker）解释了因变量（charges）变动的约73.7%**"

$$R^2 = \frac{SSR}{SST} = 1 - \frac{SSE}{SST}$$

- **SST** (Sum of Squares Total): 因变量的总变动量（相对于平均值）
- **SSR** (Sum of Squares Regression): 模型解释的变动量
- **SSE** (Sum of Squares Error): 模型未能解释的变动量（误差）

| R² 范围 | 解析 |
|--------|------|
| 0.9 以上 | 非常优秀的模型 |
| 0.7 ~ 0.9 | 良好的模型 |
| 0.5 ~ 0.7 | 一般 |
| 0.5 以下 | 解释力不足 |

- 我们模型的 R² = 0.74 属于**"良好"水平**
- **剩余的26.3%**由当前模型中未包含的因素来解释：
  - 基础疾病有无、居住地区、职业、运动习惯等
  - 或变量间的非线性关系（例如：吸烟者的BMI效应可能与非吸烟者不同）

## 9. 回归系数 (Coefficients)

```
age           264.799803
sex            17.344661
bmi           297.514806
children      469.339602
smoker      23469.280173
dtype: float64
```

### 各变量影响力解析

| 变量 | 系数 | 解析 |
|------|------|------|
| **smoker** | **+23,469.28** | **影响最大**。吸烟者(1)相比非吸烟者(0)保险费高约 $23,469 |
| **children** | +469.34 | 每增加1个子女，保险费增加约 $469 |
| **bmi** | +297.51 | BMI每增加1，保险费增加约 $298 |
| **age** | +264.80 | 年龄每增加1岁，保险费增加约 $265 |
| **sex** | +17.34 | 男性(1)相比女性(0)高约 $17 → **实际上影响微乎其微** |

### 核心洞察

1. **吸烟(smoker)是压倒性的首要因素**: 系数为23,469，比其他变量大数十至数百倍。仅吸烟与否一项就是保险费最大的决定因素。
2. **性别(sex)几乎无意义**: 系数仅为17，实际影响可以忽略。相比性别，健康相关变量对保险费重要得多。
3. **BMI和年龄影响相近**: 分别为298和265，影响程度相似。
4. **子女数也有显著影响**: 每增加1个子女约增加 $469 → 反映了根据受抚养人数确定保险费的机制。

## 10. 截距 (Intercept)

```
-11576.999976112374
```

### 解析

- **截距(intercept) = -$11,577**
- 这表示公式中**所有自变量为0时的charges值**
- 实际上不存在"0岁、女性、BMI为0、无子女、非吸烟者"这样的人，因此与其赋予截距现实意义，不如将其理解为**数学上的校正值**更为恰当
- 截距为负的原因：由于其他系数之和为较大的正数，截距需要为负数才能使整体预测值落在合理范围内

## 11. 最终预测公式

$$charges = 264.80 \times age + 17.34 \times sex + 297.51 \times bmi + 469.34 \times children + 23469.28 \times smoker - 11577.00$$

### 预测示例

**30岁，男性，BMI 25，1个子女，非吸烟者**的预测保险费：

$$charges = 264.80 \times 30 + 17.34 \times 1 + 297.51 \times 25 + 469.34 \times 1 + 23469.28 \times 0 - 11577.00$$
$$= 7944.00 + 17.34 + 7437.75 + 469.34 + 0 - 11577.00$$
$$= \$4,291.43$$

**同等条件的吸烟者**：

$$charges = 4291.43 + 23469.28 = \$27,760.71$$

→ 仅凭吸烟与否一项，保险费就相差**约6.5倍**！

## 12. 综合评价

### 模型性能总结

| 指标 | 值 | 评价 |
|------|------|------|
| RMSE | $5,684.93 | 相对于平均保险费约43%的误差 |
| R² | 0.7368 | 良好（73.7%解释力） |

### 模型的优势
- 解释性非常直观：可以定量把握各变量对保险费的影响
- 训练速度非常快
- 明确捕捉到吸烟是保险费的核心决定因素

### 模型的局限
- 对吸烟者内部的精细预测不足（未捕捉非线性关系）
- 由于charges呈右偏(right-skewed)分布，在高额保险费区间预测误差增大
- 未能考虑交互效应(interaction effect)（例如：吸烟者的BMI效应可能更大）

### 改进方向
1. **对数变换**: 对charges取log使分布正态化
2. **添加交互项**: 添加smoker × bmi等交互作用项
3. **非线性模型**: 使用决策树、随机森林、XGBoost等捕捉非线性关系
4. **正则化技术**: 通过Ridge、Lasso、Elastic Net提升泛化性能
