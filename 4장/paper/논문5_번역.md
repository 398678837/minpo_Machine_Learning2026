# 논문 5 번역 및 상세 해설: Variable Selection via Nonconcave Penalized Likelihood and its Oracle Properties

> Fan, J., & Li, R. (2001). Journal of the American Statistical Association, 96(456), 1348-1360.

---

## 1. 서론 (Introduction) 해설

### 변수 선택의 중요성

통계 모델링에서 변수 선택(variable selection)은 두 가지 목적을 가진다:
1. **예측 정확도 향상**: 불필요한 변수를 제거하면 과적합이 줄어들고 예측이 개선된다
2. **해석 가능성**: 적은 수의 변수로 현상을 설명하면 모델의 이해와 소통이 용이하다

### 기존 방법들의 한계

**부분집합 선택 (Subset Selection):**
- 장점: 해석 가능한 희소 모델
- 단점: 계산 비용이 지수적($2^p$), 불안정(데이터의 작은 변화에 민감)

**Lasso (L1 패널티):**
- 장점: 볼록 최적화, 희소 해, 계산 효율적
- 단점: 큰 계수를 과도하게 축소 -> 편향 문제, 오라클 성질 미달

### 본 논문의 목표

> "A good penalty function should result in an estimator with three properties: unbiasedness, sparsity, and continuity."

"좋은 패널티 함수는 세 가지 성질을 가진 추정량을 만들어야 한다: 비편향성, 희소성, 연속성."

저자들은 이 세 성질을 동시에 만족하는 패널티가 반드시 비볼록이어야 함을 보이고, 구체적인 예로 SCAD를 제안한다.

---

## 2. 패널티 함수의 조건 (Properties of Good Penalty Functions)

### 세 가지 필수 조건

**조건 1: 비편향성 (Unbiasedness)**
큰 계수에 대해 추정량의 편향이 거의 없어야 한다. 즉, 충분히 큰 $|\beta_j|$에 대해 패널티의 영향이 무시할 수 있어야 한다.

$$p'_\lambda(|\beta|) = 0 \quad \text{for sufficiently large } |\beta|$$

**조건 2: 희소성 (Sparsity)**
작은 계수를 정확히 0으로 만들어야 한다. 이는 연성 임계값(soft thresholding)과 유사한 효과를 가져야 함을 의미한다.

**조건 3: 연속성 (Continuity)**
추정량이 데이터의 연속 함수여야 한다. 이는 모델의 안정성을 보장하며, 데이터의 미세한 변화에 의해 추정량이 급변하지 않아야 함을 의미한다.

### 핵심 관찰

L1 패널티($p_\lambda(\beta) = \lambda|\beta|$)의 미분은 $p'_\lambda(\beta) = \lambda$로 상수이다. 이는 $\beta$의 크기에 관계없이 동일한 양만큼 축소함을 의미하므로, 조건 1(비편향성)을 만족하지 못한다.

경성 임계값(hard thresholding)의 패널티는 조건 1과 2를 만족하지만 조건 3(연속성)을 만족하지 못한다.

따라서 세 조건을 모두 만족하려면, 패널티의 미분이 $|\beta|$가 커짐에 따라 감소하여 궁극적으로 0이 되어야 한다. 이러한 패널티는 반드시 비볼록(nonconvex)이다.

## 3. SCAD 패널티 (Smoothly Clipped Absolute Deviation)

### 정의

SCAD 패널티는 그 미분(derivative)으로 정의된다:

$$p'_\lambda(\theta) = \lambda \left\{ I(\theta \leq \lambda) + \frac{(a\lambda - \theta)_+}{(a-1)\lambda} I(\theta > \lambda) \right\}, \quad \theta > 0$$

이를 적분하면 패널티 함수 자체는:

$$p_\lambda(\theta) = \begin{cases} \lambda|\theta| & \text{if } |\theta| \leq \lambda \\ \frac{-\theta^2 + 2a\lambda|\theta| - \lambda^2}{2(a-1)} & \text{if } \lambda < |\theta| \leq a\lambda \\ \frac{(a+1)\lambda^2}{2} & \text{if } |\theta| > a\lambda \end{cases}$$

### 매개변수 해석

- **$\lambda$**: 정규화 강도. 크면 더 많은 변수가 제거됨
- **$a$**: 패널티가 상수가 되는 지점을 결정. 보통 $a = 3.7$ 사용 (베이지안 위험 분석과 BIC에 의해 권장)

### 세 구간의 의미

1. **$|\theta| \leq \lambda$** (작은 계수):
   - L1(Lasso)과 동일한 패널티
   - 이 구간의 계수들은 0으로 축소됨 (변수 제거)

2. **$\lambda < |\theta| \leq a\lambda$** (중간 계수):
   - 패널티의 기울기가 점차 감소
   - 축소의 정도가 계수의 크기에 반비례하여 줄어듬
   - 이 전이 구간이 "smoothly clipped"의 의미

3. **$|\theta| > a\lambda$** (큰 계수):
   - 패널티가 상수 -> 미분이 0
   - 추가적인 축소가 없음 -> 편향 없음
   - 큰 효과를 가진 변수의 계수가 보존됨

## 4. 오라클 성질의 증명 (Oracle Properties)

### 오라클의 정의

"오라클"은 진정한 모델 $M^* = \{j : \beta_j^* \neq 0\}$을 사전에 알고 있는 가상의 존재이다. 오라클이 있다면, 중요한 변수만으로 OLS를 수행하여 최적의 추정을 할 수 있다.

### 오라클 추정량

$$\hat{\beta}^{oracle}_{M^*} = (X_{M^*}^TX_{M^*})^{-1}X_{M^*}^Ty$$

여기서 $X_{M^*}$는 진정으로 중요한 변수들의 열(column)만 포함하는 부분 설계 행렬이다.

### 정리 (Theorem 2): SCAD의 오라클 성질

적절한 정칙 조건 하에서, SCAD 패널티 추정량 $\hat{\beta}$는 다음을 만족한다:

**(a) 희소성 (Sparsity):**
$$P(\hat{\beta}_j = 0, \forall j \notin M^*) \to 1 \quad \text{as } n \to \infty$$

진정으로 0인 계수는 확률 1로 정확히 0으로 추정된다.

**(b) 점근 정규성 (Asymptotic Normality):**
$$\sqrt{n}(\hat{\beta}_{M^*} - \beta^*_{M^*}) \xrightarrow{d} N(0, \Sigma^*)$$

여기서 $\Sigma^*$는 오라클 추정량의 점근 공분산과 동일하다.

### Lasso와의 비교

Lasso(L1)는 (a)를 만족할 수 있지만(일관성 있는 변수 선택, 특정 조건 하), (b)를 만족하지 못한다. Lasso는 큰 계수를 $\lambda$만큼 일률적으로 축소하므로, 점근적 편향이 0이 되지 않는다. 이것이 Lasso의 근본적 한계이다.

## 5. 계산 알고리즘: LLA (Local Linear Approximation)

### 문제점

SCAD는 비볼록 패널티이므로, 목적 함수가 여러 개의 지역 최솟값(local minima)을 가질 수 있다. 전역 최적해를 찾는 것이 일반적으로 어렵다.

### LLA 알고리즘

Fan과 Li는 **국소 선형 근사(LLA)** 알고리즘을 제안한다:

1. 초기값 $\hat{\beta}^{(0)}$ 설정 (예: Lasso 해 또는 OLS 해)
2. 반복 $t = 0, 1, 2, \ldots$:

$$\hat{\beta}^{(t+1)} = \arg\min_\beta \left\{ \|y - X\beta\|^2 + n \sum_{j=1}^{p} p'_\lambda(|\hat{\beta}_j^{(t)}|) |\beta_j| \right\}$$

이는 SCAD 패널티를 현재 추정치에서의 **가중 L1 패널티**로 근사하는 것이다. 각 반복에서 풀어야 하는 문제는 **가중 Lasso** 문제이므로, 좌표 하강법으로 효율적으로 풀 수 있다.

### LLA의 장점

- 각 단계가 볼록 문제 -> 효율적으로 풀 수 있음
- 수렴이 빠름 (보통 2-3회 반복으로 충분)
- 좋은 초기값(Lasso 해)을 사용하면 좋은 지역 최적해에 도달할 가능성이 높음

## 6. 시뮬레이션 연구

### 설정

$n = 100$ 관측치, 다양한 $p$ 값, 진정한 비영 계수의 수가 소수인 희소 설정

### 비교 방법

1. SCAD 패널티 추정량
2. Lasso (L1 패널티)
3. 경성 임계값 (Hard thresholding)
4. 부분집합 선택 (Best subset selection)
5. OLS (정규화 없음)

### 주요 결과

| 방법 | 모델 오류 | 올바른 적합 비율 |
|------|-----------|-----------------|
| SCAD | 가장 작음 | 가장 높음 (95%+) |
| Lasso | 중간 | 높음 (80-90%) |
| 경성 임계값 | 작음 | 중간 (불안정) |
| 부분집합 선택 | 작음 | 높음 (계산 비용 높음) |
| OLS | 가장 큼 | N/A |

SCAD는 가장 작은 모델 오류와 가장 높은 올바른 모델 식별 비율을 달성했다.

## 8. 적응적 Lasso와의 연결 (Adaptive Lasso)

Zou(2006)는 Fan & Li(2001)의 아이디어에 영감을 받아 **적응적 Lasso(Adaptive Lasso)**를 제안했다:

$$\hat{\beta}^{alasso} = \arg\min_\beta \left\{ \|y - X\beta\|^2 + \lambda \sum_{j=1}^{p} \hat{w}_j |\beta_j| \right\}$$

여기서 $\hat{w}_j = 1/|\hat{\beta}_j^{init}|^\gamma$ ($\gamma > 0$)로, 초기 추정치가 큰 변수에 더 작은 패널티를 부여한다.

적응적 Lasso는:
- 오라클 성질을 가짐 (Fan & Li의 이론적 프레임워크에서 증명 가능)
- 볼록 최적화 문제 (SCAD와 달리)
- LLA 알고리즘의 1회 반복과 동치

## 9. 핵심 인사이트 (Core Insights)

1. **편향과 선택의 딜레마**: L1 패널티는 변수 선택과 계수 추정을 하나의 매개변수로 동시에 제어하려 하므로, 하나를 잘하면 다른 하나가 나빠지는 딜레마가 발생한다. SCAD는 작은 계수와 큰 계수를 다르게 처리하여 이 딜레마를 해결한다.

2. **오라클 성질의 의미**: 오라클 성질은 "데이터가 충분히 많으면, 패널티 추정량이 진정한 모델을 아는 것과 동일한 성능을 달성한다"는 것이다. 이는 변수 선택의 궁극적 목표를 수학적으로 정의한 것이다.

3. **비볼록의 불가피성**: 비편향성, 희소성, 연속성을 동시에 만족하려면 비볼록 패널티가 필수적이다. 이는 볼록 최적화의 편리함과 통계적 최적성 사이의 트레이드오프를 보여준다.

4. **실용적 교훈**: Lasso가 만능은 아니다. 큰 계수의 편향 문제가 중요한 경우(예: 효과 크기의 정확한 추정이 필요한 의학/경제학 연구), SCAD나 적응적 Lasso 같은 대안을 고려해야 한다.

5. **이론과 실제의 간극**: SCAD는 이론적으로 우수하지만, 비볼록 최적화의 어려움 때문에 실무에서는 Lasso가 더 널리 사용된다. 이론적 최적성과 계산적 편의성 사이의 균형을 이해하는 것이 중요하다.
