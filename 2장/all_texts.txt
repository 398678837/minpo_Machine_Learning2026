T|Impact of float precision on ML
T|IEEE 754 Floating Point Limits and Dealing with them
T|# IEEE 754 Floating Point Limitations
T|print(0.1 + 0.2)         # 0.30000000000000004
T|print(0.1 + 0.2 == 0.3)  # False !!
T|# How to Compare Correctly
T|import math
T|print(math.isclose(0.1 + 0.2, 0.3))  # True
T|Why Precision is Important in ML
T|float32 Precision ≈ 10⁻⁷
T|float64 Precision ≈ 10⁻¹⁶
T|Gradient descent: Vanishing gradient in float32 when η=10⁻⁶
T|Loss function: If p→0 in log(p), -∞ divergence
T|→ Requires np.clip(p, 1e-15, 1)
T|Mixed precision: GPU optimization with float16 + float32
T|Formula: float32 precision ≈ 10⁻⁷, float64 precision ≈ 10⁻¹⁶

T|SECTION 4
T|sklearn's fit/predict/transform patterns
T|Estimator API - consistent interface for all models
T|class SimpleScaler:
T|    def __init__(self, feature_range=(0, 1)):
T|        self.feature_range = feature_range
T|        self.min_ = None
T|        self.max_ = None
T|def fit(self, X): # Learn statistics from training data
T|        self.min_ = min(X)
T|        self.max_ = max(X)
T|        return self
T|def transform(self, (Convenience)
T|        range_ = self.max_ - self.min_
T|        lo, hi = self.feature_range
T|        return [(x-self.min_)/range_*(hi-lo)+lo
T|                for x in X]
T|Important: fit() must only be used on training data!
T|Calling fit() on test data = data leakage
T|호출 시점
T|fit(X)
T|학습 데이터에서 파라미터 학습
T|학습 시
T|predict(X)
T|학습된 모델로 예측 수행
T|추론 시
T|transform(X)
T|학습된 파라미터로 데이터 변환
T|학습/추론 시
T|fit_transform(X)
T|fit + transform 한 번에
T|학습 시 (편의)
T|중요: fit()은 반드시 학습 데이터에서만!
T|테스트 데이터에 fit() 호출 = 데이터 누출(data leakage)

T|def euclidean_distance(a, b):
T|"""Euclidean distance calculation"""
T|    return math.sqrt(
T|        sum((ai-bi)**2 for ai,bi in zip(a,b)))
T|def knn_predict(X_train, y_train, x_query, k=5):
T|"""KNN prediction: Majority vote of k nearest neighbors"""
T|    distances = [
T|        (euclidean_distance(x_query, x_tr), label)
T|        for x_tr, label
T|        in zip(X_train, y_train)]
T|    distances.sort(key=lambda x: x[0])
T|    k_nearest = distances[:k]
T|    votes = [label for _, label in k_nearest]
T|    return max(set(votes), key=votes.count)
