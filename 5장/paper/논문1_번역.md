# 논문 1 상세 해설: The Regression Analysis of Binary Sequences

> **원저**: D. R. Cox (1958), "The Regression Analysis of Binary Sequences", *Journal of the Royal Statistical Society, Series B*, Vol. 20, No. 2, pp. 215-242

---

## 1. 서론 (Introduction)

### 문제 정의

Cox는 논문의 서두에서 이진 반응(binary response)에 대한 회귀 분석 문제를 정의한다. 전통적인 회귀 분석은 연속형 종속 변수(continuous dependent variable)를 다루지만, 많은 실제 문제에서 결과 변수는 이진(binary)으로 관측된다:

- **의학**: 환자의 치료 성공/실패
- **독성학**: 특정 용량에서 동물의 생존/사망
- **사회과학**: 유권자의 찬성/반대

이러한 이진 결과를 설명 변수의 함수로 모델링하는 것이 논문의 핵심 목적이다.

### 기존 접근법의 한계

1958년 이전에는 이진 데이터에 대해 주로 **프로빗 분석(probit analysis)**이 사용되었다. 프로빗 모델은 정규분포의 누적분포함수(CDF)를 사용하여 확률을 모델링한다:

$$P(Y=1|X) = \Phi(\beta_0 + \beta_1 X)$$

여기서 $\Phi$는 표준정규분포의 CDF이다. 이 접근법은 통계적으로 유효하지만, 파라미터의 해석이 직관적이지 않다는 한계가 있었다.

---

## 2. 로짓 모델의 제안 (The Logit Model)

### 로짓 변환의 동기

Cox는 확률을 직접 선형 함수로 모델링하면 $P(Y=1|X) = \beta_0 + \beta_1 X$ 형태가 되는데, 이 경우 예측 확률이 0보다 작거나 1보다 큰 값을 가질 수 있다는 근본적 문제를 지적한다.

이를 해결하기 위해 Cox는 **로짓 변환(logit transformation)**을 제안한다:

$$\text{logit}(p) = \log\left(\frac{p}{1-p}\right)$$

### 모델 정의

로짓 모델(logit model)은 다음과 같이 정의된다:

$$\log\left(\frac{p_i}{1-p_i}\right) = \mathbf{x}_i^T \boldsymbol{\beta} = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_k x_{ik}$$

이를 확률의 관점에서 다시 쓰면:

$$p_i = P(Y_i = 1 | \mathbf{x}_i) = \frac{e^{\mathbf{x}_i^T \boldsymbol{\beta}}}{1 + e^{\mathbf{x}_i^T \boldsymbol{\beta}}} = \frac{1}{1 + e^{-\mathbf{x}_i^T \boldsymbol{\beta}}}$$

이것이 바로 **시그모이드 함수(sigmoid function)**이다.

### 로짓 변환의 핵심 성질

1. **범위 보장**: $p \in (0, 1)$일 때 $\text{logit}(p) \in (-\infty, +\infty)$ -> 선형 결합과 범위가 일치
2. **대칭성**: $\text{logit}(p) = -\text{logit}(1-p)$ -> 클래스 레이블을 바꿔도 대칭적 구조 유지
3. **오즈비 해석**: 회귀 계수의 지수 변환이 오즈비를 제공

## 3. 파라미터 추정 (Parameter Estimation)

### 우도 함수 (Likelihood Function)

$n$개의 독립적인 이진 관측치 $(y_1, y_2, \ldots, y_n)$이 주어졌을 때, 우도 함수는:

$$L(\boldsymbol{\beta}) = \prod_{i=1}^{n} p_i^{y_i} (1-p_i)^{1-y_i}$$

여기서 $p_i = \sigma(\mathbf{x}_i^T \boldsymbol{\beta})$이다.

### 로그 우도 함수 (Log-Likelihood Function)

계산의 편의를 위해 로그 우도를 사용한다:

$$\ell(\boldsymbol{\beta}) = \sum_{i=1}^{n} \left[ y_i \log(p_i) + (1-y_i) \log(1-p_i) \right]$$

**핵심 관찰**: 이 로그 우도의 음수(negative)를 취하고 $n$으로 나누면, 바로 5장에서 배우는 **이진 교차 엔트로피(Binary Cross Entropy) / 로그 손실(Log Loss)**이 된다:

$$J(\boldsymbol{\beta}) = -\frac{1}{n}\ell(\boldsymbol{\beta}) = -\frac{1}{n}\sum_{i=1}^{n} \left[ y_i \log(p_i) + (1-y_i) \log(1-p_i) \right]$$

즉, **MLE로 파라미터를 추정하는 것 = 로그 손실을 최소화하는 것**이다.

### 스코어 방정식 (Score Equations)

로그 우도를 $\boldsymbol{\beta}$에 대해 미분하면 스코어 함수를 얻는다:

$$\frac{\partial \ell}{\partial \beta_j} = \sum_{i=1}^{n} (y_i - p_i) x_{ij} = 0, \quad j = 0, 1, \ldots, k$$

이것을 행렬 형태로 쓰면:

$$\mathbf{X}^T (\mathbf{y} - \mathbf{p}) = \mathbf{0}$$

**핵심 관찰**: 이 스코어 방정식의 좌변은 바로 5장에서 배우는 **그래디언트(gradient)**이다. 경사하강법에서 파라미터를 업데이트할 때 사용하는 그래디언트가 이 스코어 함수에서 유도된다.

### 뉴턴-랩슨 방법 (Newton-Raphson Method)

스코어 방정식은 비선형이므로 닫힌 형태의 해(closed-form solution)가 존재하지 않는다. Cox는 반복적 수치 방법, 특히 뉴턴-랩슨(Newton-Raphson) 방법을 제안한다:

$$\boldsymbol{\beta}^{(t+1)} = \boldsymbol{\beta}^{(t)} - \left[\frac{\partial^2 \ell}{\partial \boldsymbol{\beta} \partial \boldsymbol{\beta}^T}\right]^{-1} \frac{\partial \ell}{\partial \boldsymbol{\beta}}$$

헤시안 행렬(Hessian matrix)은:

$$\frac{\partial^2 \ell}{\partial \boldsymbol{\beta} \partial \boldsymbol{\beta}^T} = -\mathbf{X}^T \mathbf{W} \mathbf{X}$$

여기서 $\mathbf{W} = \text{diag}(p_i(1-p_i))$는 가중치 대각 행렬이다.

이것이 바로 **반복 재가중 최소제곱법(Iteratively Reweighted Least Squares, IRLS)**의 원형이며, sklearn의 `LogisticRegression`에서 `solver='lbfgs'`가 이 방법의 개선된 버전을 사용한다.

## 4. 오즈비 해석 (Odds Ratio Interpretation)

### 단일 이진 설명 변수의 경우

설명 변수 $X$가 0 또는 1의 값만 취하는 경우:

- $X=1$일 때의 오즈: $\frac{p_1}{1-p_1} = e^{\beta_0 + \beta_1}$
- $X=0$일 때의 오즈: $\frac{p_0}{1-p_0} = e^{\beta_0}$

따라서 **오즈비(Odds Ratio)**는:

$$OR = \frac{e^{\beta_0 + \beta_1}}{e^{\beta_0}} = e^{\beta_1}$$

### 연속형 설명 변수의 경우

설명 변수 $X$가 1단위 증가할 때:

$$\frac{\text{Odds}(X+1)}{\text{Odds}(X)} = \frac{e^{\beta_0 + \beta_1(X+1)}}{e^{\beta_0 + \beta_1 X}} = e^{\beta_1}$$

즉, **$X$가 1단위 증가할 때마다 오즈가 $e^{\beta_1}$배 변한다**.

- $\beta_1 > 0$: $e^{\beta_1} > 1$ -> 오즈 증가 (양성 확률 증가)
- $\beta_1 = 0$: $e^{\beta_1} = 1$ -> 오즈 불변 (해당 변수 영향 없음)
- $\beta_1 < 0$: $e^{\beta_1} < 1$ -> 오즈 감소 (양성 확률 감소)

이 오즈비 해석이 5장에서 `model.coef_`를 해석하는 핵심 근거이다.

## 5. 적합도 검정 (Goodness of Fit)

### 우도비 검정 (Likelihood Ratio Test)

Cox는 모델의 적합도를 평가하기 위해 우도비 검정을 제안한다. 두 모델(귀무 모델과 대립 모델)의 로그 우도를 비교:

$$G^2 = -2(\ell_0 - \ell_1)$$

여기서 $\ell_0$은 축소 모델(reduced model)의 로그 우도, $\ell_1$은 완전 모델(full model)의 로그 우도이다. 대표본에서 $G^2$는 카이제곱 분포를 따른다.

### 이탈도 (Deviance)

모델의 전반적 적합도를 나타내는 이탈도(deviance):

$$D = -2[\ell(\hat{\boldsymbol{\beta}}) - \ell_{\text{saturated}}]$$

포화 모델(saturated model)은 각 관측치에 대해 별도의 파라미터를 가지는 모델이다.

## 6. 프로빗 모델과의 비교 (Comparison with Probit)

Cox는 로짓 모델과 프로빗 모델을 상세히 비교한다:

| 측면 | 로짓 (Logit) | 프로빗 (Probit) |
|------|-------------|----------------|
| **링크 함수** | $\log(p/(1-p))$ | $\Phi^{-1}(p)$ |
| **기반 분포** | 로지스틱 분포 | 정규분포 |
| **꼬리 행동** | 더 두꺼운 꼬리 | 더 얇은 꼬리 |
| **해석** | 오즈비로 직관적 해석 | 직접적 해석 어려움 |
| **실용적 차이** | 대부분의 경우 거의 동일 | 대부분의 경우 거의 동일 |

Cox는 두 모델이 실용적으로 거의 동일한 결과를 제공하지만, **오즈비를 통한 직관적 해석**이 가능하다는 점에서 로짓 모델의 우위를 주장하였다.

## 7. 다중 설명 변수로의 확장 (Extension to Multiple Covariates)

단일 설명 변수에서 다중 설명 변수로의 확장은 자연스럽다:

$$\text{logit}(p_i) = \beta_0 + \beta_1 x_{i1} + \beta_2 x_{i2} + \cdots + \beta_k x_{ik}$$

Cox는 이 확장에서 주의해야 할 점들을 논의한다:

1. **다중공선성(Multicollinearity)**: 설명 변수 간 높은 상관관계가 있으면 파라미터 추정이 불안정해짐
2. **교호작용(Interaction)**: 설명 변수 간 교호작용 항을 포함할 수 있음
3. **모델 선택(Model Selection)**: 어떤 설명 변수를 포함할 것인가의 문제

## 8. 핵심 수식 총정리

| 개념 | 수식 |
|------|------|
| 로짓 변환 | $\text{logit}(p) = \log\left(\frac{p}{1-p}\right)$ |
| 시그모이드 함수 | $\sigma(z) = \frac{1}{1+e^{-z}}$ |
| 모델 | $p = \sigma(\mathbf{x}^T\boldsymbol{\beta})$ |
| 로그 우도 | $\ell = \sum_{i} [y_i \log p_i + (1-y_i)\log(1-p_i)]$ |
| 그래디언트 | $\nabla_{\beta}\ell = \mathbf{X}^T(\mathbf{y}-\mathbf{p})$ |
| 헤시안 | $\nabla^2_{\beta}\ell = -\mathbf{X}^T\mathbf{W}\mathbf{X}$ |
| 오즈비 | $OR = e^{\beta_j}$ |

## 9. 논문의 역사적 의의

Cox의 이 1958년 논문은 통계학 역사에서 가장 영향력 있는 논문 중 하나로 평가된다:

1. **로지스틱 회귀의 공식적 정립**: 이전에 단편적으로 사용되던 로짓 모델을 체계적으로 정리
2. **일반화선형모형(GLM)의 선구**: 후에 Nelder & Wedderburn(1972)이 GLM을 정립하는 데 핵심적 영감을 제공
3. **기계학습과의 연결**: 현대 기계학습에서 로지스틱 회귀는 가장 기본적인 분류 알고리즘으로 자리매김
4. **딥러닝의 기초**: 신경망의 출력층에서 사용하는 시그모이드/소프트맥스 활성화 함수와 크로스 엔트로피 손실 함수가 모두 이 논문에서 유래

이 논문을 통해, 5장에서 배우는 로지스틱 회귀의 모든 수학적 기초가 1958년에 이미 정립되었음을 확인할 수 있다.
