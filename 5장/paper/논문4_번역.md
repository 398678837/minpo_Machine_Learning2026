# 논문 4 상세 해설: On Discriminative vs. Generative classifiers

> **원저**: A. Y. Ng & M. I. Jordan (2002), "On Discriminative vs. Generative classifiers: A comparison of logistic regression and naive Bayes", *NIPS 2001*

---

## 1. 서론: 판별 모델 vs 생성 모델

### 1.1 두 가지 근본적 접근법

기계학습에서 분류(classification) 문제를 푸는 두 가지 근본적 접근법:

**판별 접근법 (Discriminative Approach)**:
- 조건부 분포 $P(y|\mathbf{x})$를 직접 모델링
- "이 입력이 어떤 클래스에 속하는가?"에 직접 답변
- 입력 공간의 결정 경계(decision boundary)를 직접 학습
- 대표 모델: 로지스틱 회귀, SVM, 결정 트리, 신경망

**생성 접근법 (Generative Approach)**:
- 클래스별 조건부 분포 $P(\mathbf{x}|y)$와 사전 확률 $P(y)$를 모델링
- 베이즈 정리: $P(y|\mathbf{x}) \propto P(\mathbf{x}|y)P(y)$
- "각 클래스가 데이터를 어떻게 생성하는가?"를 학습
- 대표 모델: 나이브 베이즈, 가우시안 판별 분석(GDA), HMM

### 1.2 논문의 핵심 질문

> "판별 분류기와 생성 분류기 중 어떤 것을 사용해야 하는가? 그 선택은 데이터의 양과 어떤 관계가 있는가?"

## 2. 이론적 분석: 나이브 베이즈와 로지스틱 회귀의 관계

### 2.1 나이브 베이즈 모델

나이브 베이즈의 핵심 가정: **조건부 독립성**

$$P(\mathbf{x}|y) = \prod_{j=1}^{d} P(x_j|y)$$

이진 분류($y \in \{0, 1\}$)에서 각 특성 $x_j$가 지수족(exponential family)에 속하면:

$$P(x_j|y=k) = c_j(x_j) \exp(\eta_{jk} t_j(x_j) - a_{jk})$$

### 2.2 나이브 베이즈에서 로지스틱 회귀로의 유도

로그 오즈비(log-odds ratio)를 계산하면:

$$\log\frac{P(y=1|\mathbf{x})}{P(y=0|\mathbf{x})} = \log\frac{P(y=1)}{P(y=0)} + \sum_{j=1}^{d}\log\frac{P(x_j|y=1)}{P(x_j|y=0)}$$

특성이 지수족에 속할 때:

$$\log\frac{P(x_j|y=1)}{P(x_j|y=0)} = (\eta_{j1} - \eta_{j0})t_j(x_j) - (a_{j1} - a_{j0})$$

이를 정리하면:

$$\log\frac{P(y=1|\mathbf{x})}{P(y=0|\mathbf{x})} = \underbrace{\left[\log\frac{P(y=1)}{P(y=0)} - \sum_j(a_{j1}-a_{j0})\right]}_{\theta_0} + \sum_{j=1}^{d} \underbrace{(\eta_{j1}-\eta_{j0})}_{\theta_j} t_j(x_j)$$

이것이 바로 **로지스틱 회귀의 형태**이다:

$$P(y=1|\mathbf{x}) = \frac{1}{1 + \exp(-\theta_0 - \sum_j \theta_j t_j(x_j))}$$

### 2.3 핵심 통찰: 같은 모델, 다른 추정

나이브 베이즈와 로지스틱 회귀는 동일한 **가설 공간(hypothesis space)**을 공유한다. 차이는 파라미터를 추정하는 방법에 있다:

| 측면 | 나이브 베이즈 | 로지스틱 회귀 |
|------|:------------:|:------------:|
| **목적 함수** | 결합 우도 $P(\mathbf{x}, y)$ 최대화 | 조건부 우도 $P(y|\mathbf{x})$ 최대화 |
| **추정 대상** | $P(x_j|y)$의 파라미터들 개별 추정 | $\theta_0, \theta_1, \ldots, \theta_d$ 직접 추정 |
| **파라미터 수** | 모델 가정에 의해 제한 | 자유 파라미터 $d+1$개 |
| **제약 조건** | 나이브 베이즈 가정이 파라미터 공간을 제한 | 파라미터 공간에 대한 제약 없음 |

## 3. 수렴 속도 분석

### 3.1 정리 1: 나이브 베이즈의 수렴 속도

> **정리**: 나이브 베이즈의 파라미터 추정 오차는 $O(\log n / n)$의 속도로 0에 수렴한다. 즉, 점근적 성능에 도달하는 데 필요한 표본 수는 $O(\log n)$이다.

**직관적 이해**: 나이브 베이즈는 각 특성의 분포를 개별적으로 추정한다. 각 분포의 파라미터 추정에 필요한 표본 수는 상수이므로, 전체적으로 $O(\log d)$ (특성 수에 로그적으로 비례) 표본이면 충분하다.

### 3.2 정리 2: 로지스틱 회귀의 수렴 속도

> **정리**: 로지스틱 회귀의 파라미터 추정 오차는 $O(d/n)$의 속도로 0에 수렴한다. 점근적 성능에 도달하는 데 필요한 표본 수는 $O(d)$이다.

**직관적 이해**: 로지스틱 회귀는 $d+1$개의 파라미터를 동시에 추정한다. 각 파라미터를 정확히 추정하려면 상수 개의 표본이 필요하므로, 전체적으로 $O(d)$ 표본이 필요하다.

### 3.3 핵심 비교

나이브 베이즈의 가정이 **정확히** 성립하는 경우:

| 표본 수 $n$ | 나이브 베이즈 오차 | 로지스틱 회귀 오차 | 우수한 모델 |
|:-----------:|:-----------------:|:-----------------:|:-----------:|
| 매우 적음 | 높음 (하지만 빠르게 감소) | 매우 높음 | 나이브 베이즈 |
| 적음 | 점근적 수준에 근접 | 여전히 높음 | 나이브 베이즈 |
| 전환점 | 점근적 수준 | 나이브 베이즈와 동일 | 동일 |
| 많음 | 점근적 수준 (편향 존재 가능) | 점근적 수준 이하 | 로지스틱 회귀 |
| 매우 많음 | 점근적 오차에 수렴 | 0에 수렴 | 로지스틱 회귀 |

### 3.4 그래프로 표현

```
  오차(Error)
  ^
  |  *
  |   *
  |    *  . . . . . . . . . . . . .  <- 나이브 베이즈 점근적 오차
  |     * .                            (모델 가정이 틀리면 편향 존재)
  |      *.
  |       *
  |      . *
  |     .   *
  |    .     *
  |   .       *
  |  .         *  *  *  *  *  *  -> 0  <- 로지스틱 회귀
  | .
  +-----------------------------------------> 표본 수(n)
       ^
       전환점 (crossover point)

  . = 나이브 베이즈
  * = 로지스틱 회귀
```

## 4. 점근적 성능 분석

### 4.1 모델 가정이 맞는 경우

나이브 베이즈의 가정이 정확히 성립하면:
- 두 모델의 점근적 오차가 동일
- 나이브 베이즈가 더 빠르게 이 오차에 도달
- 따라서 **유한 표본에서는 나이브 베이즈가 항상 같거나 우수**

### 4.2 모델 가정이 틀린 경우 (실제로 더 일반적)

나이브 베이즈의 가정(조건부 독립성)이 위반되면:
- 나이브 베이즈에 점근적 편향(asymptotic bias)이 존재
- 로지스틱 회귀는 여전히 일치추정량(consistent)
- **대표본에서 로지스틱 회귀가 확실히 우수**
- 소표본에서는 여전히 나이브 베이즈가 우수할 수 있음 (편향-분산 트레이드오프)

### 4.3 정리 3 (비공식적 요약)

> 나이브 베이즈의 가정이 위반될 때에도, 충분히 작은 표본에서는 나이브 베이즈가 로지스틱 회귀보다 우수할 수 있다. 이는 나이브 베이즈의 **높은 편향이 낮은 분산으로 보상**되기 때문이다.

이것이 바로 **편향-분산 트레이드오프(bias-variance tradeoff)**의 구체적 발현이다.

## 5. 실험 결과

### 5.1 실험 설계

**UCI 데이터셋**을 사용한 실험:
- 다양한 벤치마크 데이터셋 (Breast Cancer, Heart, Ionosphere 등)
- 각 데이터셋에서 학습 세트 크기를 1%에서 100%까지 변화
- 로지스틱 회귀와 나이브 베이즈의 테스트 오차 비교
- 100회 반복으로 평균 성능 산출

### 5.2 주요 발견

1. **전환점의 존재**: 거의 모든 데이터셋에서 전환점이 관측됨
2. **전환점의 위치**: 대략 파라미터 수의 $O(\log d)$배에 해당하는 표본 수에서 전환
3. **나이브 베이즈의 소표본 우수성**: 학습 데이터가 적을 때 일관되게 나이브 베이즈가 우수
4. **로지스틱 회귀의 대표본 우수성**: 학습 데이터가 충분할 때 일관되게 로지스틱 회귀가 우수

## 6. 혼합 모델 (Hybrid Approach)

### 6.1 가중 결합

Ng와 Jordan은 두 모델의 장점을 결합하는 방법도 논의한다. 학습 데이터의 양에 따라 두 모델의 예측을 가중 평균:

$$P_{hybrid}(y|\mathbf{x}) = \lambda \cdot P_{NB}(y|\mathbf{x}) + (1-\lambda) \cdot P_{LR}(y|\mathbf{x})$$

여기서 $\lambda$는 교차 검증으로 결정한다.

### 6.2 정규화된 로지스틱 회귀

다른 접근법은 로지스틱 회귀에 강한 정규화(regularization)를 적용하는 것이다:

$$\hat{\boldsymbol{\theta}} = \arg\min_{\boldsymbol{\theta}} \left[-\sum_i \log P(y_i|\mathbf{x}_i; \boldsymbol{\theta}) + \lambda \|\boldsymbol{\theta}\|^2\right]$$

강한 정규화($\lambda$가 큰 경우)는 로지스틱 회귀를 나이브 베이즈 방향으로 "당기는" 효과가 있다.

## 7. 일반화: 판별-생성 쌍 (Discriminative-Generative Pairs)

### 7.1 대응 관계

논문은 나이브 베이즈-로지스틱 회귀 쌍을 넘어, 일반적인 판별-생성 쌍의 개념을 제시한다:

| 생성 모델 | 판별 모델 |
|-----------|-----------|
| 나이브 베이즈 (이산 특성) | 로지스틱 회귀 |
| 가우시안 판별 분석 (GDA) | 로지스틱 회귀 |
| Hidden Markov Model (HMM) | Conditional Random Field (CRF) |
| 가우시안 혼합 모델 (GMM) | 조건부 혼합 모델 |

### 7.2 모든 쌍에 적용되는 원칙

> "생성 모델은 적은 데이터에서 빠르게 수렴하지만, 판별 모델은 충분한 데이터에서 더 낮은 오차에 도달한다."

이 원칙은 나이브 베이즈와 로지스틱 회귀에만 국한되지 않고, 모든 판별-생성 쌍에 적용된다.

## 8. 실용적 가이드라인

### 8.1 모델 선택 기준

| 상황 | 권장 모델 |
|------|-----------|
| 학습 데이터가 매우 적음 | 나이브 베이즈 (생성) |
| 학습 데이터가 충분함 | 로지스틱 회귀 (판별) |
| 모델 가정이 정확히 성립 | 나이브 베이즈 (효율적) |
| 모델 가정이 의심스러움 | 로지스틱 회귀 (강건) |
| 분류뿐 아니라 데이터 생성도 필요 | 생성 모델 |
| 오직 분류 성능만 중요 | 판별 모델 |

### 8.2 실전 전략

1. **표본 크기 확인**: 특성 수 $d$ 대비 표본 수 $n$의 비율 확인
2. **학습 곡선 그리기**: 표본 크기를 늘려가며 두 모델의 성능 비교
3. **전환점 식별**: 로지스틱 회귀가 나이브 베이즈를 추월하는 지점 확인
4. **정규화 활용**: 소표본에서 로지스틱 회귀를 사용해야 한다면 강한 정규화 적용

## 9. 이 논문이 기계학습 전체에 미친 영향

이 논문은 단순히 두 모델의 비교를 넘어, 기계학습의 근본적 원리에 대한 통찰을 제공했다:

1. **편향-분산 트레이드오프의 구체적 예시**: 생성 모델(높은 편향, 낮은 분산) vs 판별 모델(낮은 편향, 높은 분산)
2. **모델 복잡도와 데이터 양의 관계**: 복잡한 모델에는 더 많은 데이터가 필요하다는 원칙의 구체화
3. **딥러닝에의 영향**: GAN(Generative Adversarial Network)의 발전에도 이 판별-생성 프레임워크가 영향
4. **앙상블 학습의 정당화**: 서로 다른 특성을 가진 모델을 결합하는 것의 이론적 근거

## 10. 핵심 수식 총정리

| 개념 | 수식 |
|------|------|
| 로지스틱 회귀 | $P(y=1\|\mathbf{x}) = \sigma(\boldsymbol{\theta}^T\mathbf{x})$ |
| 나이브 베이즈 | $P(y\|\mathbf{x}) \propto P(y)\prod_j P(x_j\|y)$ |
| 나이브 베이즈 -> 로짓 | $\log\frac{P(y=1\|\mathbf{x})}{P(y=0\|\mathbf{x})} = \theta_0 + \sum_j \theta_j t_j(x_j)$ |
| 생성 모델 수렴 | 표본 $O(\log d)$개에서 점근적 성능 도달 |
| 판별 모델 수렴 | 표본 $O(d)$개에서 점근적 성능 도달 |
| 정규화된 로지스틱 회귀 | $\min_\theta [-\ell(\theta) + \lambda\|\theta\|^2]$ |
