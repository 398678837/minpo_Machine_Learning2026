# 논문 3 상세 해설: Logistic Regression in Rare Events Data

> **원저**: G. King & L. Zeng (2001), "Logistic Regression in Rare Events Data", *Political Analysis*, Vol. 9, No. 2, pp. 137-163

---

## 1. 서론: 희소 사건의 문제

### 1.1 문제의 배경

King과 Zeng는 정치학에서 빈번히 마주하는 데이터의 특성을 지적하며 논의를 시작한다. 국제 관계 연구에서 주요 관심 사건들 - 전쟁 발발, 쿠데타, 국가 간 분쟁 등 - 은 전체 관측 기간에서 매우 드물게 발생한다.

**구체적 예시**:
- 국가 쌍(dyad) 데이터: 약 30만 개의 관측치 중 분쟁 발생은 약 3,000건 (~1%)
- 내전 발생: 국가-연도 관측치 중 약 2~5%에서만 발생
- 쿠데타: 전체 관측치의 약 1% 미만

저자들은 이러한 희소 사건 데이터에 표준 로지스틱 회귀를 적용했을 때, **"모든 것이 잘 작동하는 것처럼 보이지만 실제로는 체계적인 편향이 존재한다"**는 핵심 문제를 제기한다.

### 1.2 왜 문제인가?

표준 로지스틱 회귀 모델:

$$P(Y=1|\mathbf{x}) = \frac{1}{1 + e^{-\mathbf{x}^T\boldsymbol{\beta}}}$$

이 모델의 MLE는 대표본(large sample)에서 일치추정량(consistent estimator)이다. 그러나 "대표본"의 의미는 양성 사례(Y=1)와 음성 사례(Y=0) **모두가 충분히 많아야** 한다는 것이다. 희소 사건에서는 전체 표본 크기가 크더라도, 양성 사례의 수가 적기 때문에 대표본 근사가 성립하지 않을 수 있다.

---

## 2. 편향의 이론적 분석

### 2.1 MLE의 유한표본 편향

MLE $\hat{\boldsymbol{\beta}}$의 편향을 $O(n^{-1})$까지 전개하면:

$$E(\hat{\boldsymbol{\beta}}) = \boldsymbol{\beta} + \text{bias}(\hat{\boldsymbol{\beta}}) + O(n^{-2})$$

여기서 편향 항은:

$$\text{bias}(\hat{\boldsymbol{\beta}}) = (\mathbf{X}^T\mathbf{W}\mathbf{X})^{-1}\mathbf{X}^T\mathbf{W}\boldsymbol{\xi}$$

$\boldsymbol{\xi}$의 $i$번째 원소:

$$\xi_i = 0.5 \cdot h_i \cdot Q_i$$

여기서:
- $h_i$: 해트 행렬(hat matrix) $\mathbf{H} = \mathbf{W}^{1/2}\mathbf{X}(\mathbf{X}^T\mathbf{W}\mathbf{X})^{-1}\mathbf{X}^T\mathbf{W}^{1/2}$의 대각 원소
- $Q_i = (1 - 2\hat{\pi}_i)$: 비대칭 인자

### 2.2 편향의 방향

핵심 결과는 **편향의 방향**에 관한 것이다:

1. **사건이 드물 때** ($\bar{y} \approx 0$): 대부분의 $\hat{\pi}_i$가 0에 가까우므로 $Q_i = (1-2\hat{\pi}_i) > 0$ -> 편향이 양의 방향
2. 양의 방향 편향은 **회귀 계수 $|\hat{\beta}|$의 과대추정**을 의미
3. 회귀 계수의 과대추정은 **예측 확률의 과소추정**으로 이어짐 (시그모이드 함수의 S자 형태 때문)

**직관적 이해**:
- 양성 사례가 매우 적을 때, 모델은 양성과 음성을 분리하는 결정 경계를 과도하게 밀어넣는 경향이 있다
- 이로 인해 새로운 관측치에 대한 사건 발생 확률이 과소추정된다

### 2.3 수치적 예시

King과 Zeng가 제시한 시뮬레이션 결과:

| 양성 비율 | 표본 크기 | 확률 추정 편향 |
|:---------:|:---------:|:--------------:|
| 50% | 1,000 | ~0% |
| 10% | 1,000 | ~5% 과소추정 |
| 5% | 1,000 | ~20% 과소추정 |
| 1% | 1,000 | ~40% 과소추정 |
| 1% | 10,000 | ~15% 과소추정 |
| 0.1% | 100,000 | ~30% 과소추정 |

양성 비율이 낮을수록, 그리고 표본 크기가 작을수록 편향이 심해진다.

## 3. 교정 방법 1: 사전 교정 (Prior Correction)

### 3.1 케이스-통제 설계의 필요성

희소 사건 연구에서는 효율성을 위해 **케이스-통제(case-control)** 설계를 자주 사용한다:
- 모든 양성 사례(cases)를 포함
- 음성 사례(controls)는 무작위로 일정 비율만 샘플링

이 설계에서는 표본 내 양성 비율 $\bar{y}$가 인구(population)의 실제 양성 비율 $\tau$와 다르다.

### 3.2 사전 교정 공식

케이스-통제 설계에서 절편만 영향을 받으며, 다른 계수는 일치추정량이다. 보정된 절편:

$$\hat{\beta}_0^{corrected} = \hat{\beta}_0 - \ln\left[\left(\frac{1-\tau}{\tau}\right)\cdot\left(\frac{\bar{y}}{1-\bar{y}}\right)\right]$$

**유도 과정**:

표본에서의 모델: $\text{logit}(P_s(Y=1|\mathbf{x})) = \beta_0^s + \boldsymbol{\beta}_1^T\mathbf{x}$

인구에서의 모델: $\text{logit}(P(Y=1|\mathbf{x})) = \beta_0 + \boldsymbol{\beta}_1^T\mathbf{x}$

케이스-통제 설계에서:
$$\beta_0 = \beta_0^s + \ln\left(\frac{\tau}{1-\tau}\right) - \ln\left(\frac{\bar{y}}{1-\bar{y}}\right)$$

### 3.3 실용적 적용

```
예시: 신용카드 사기 탐지
- 인구 사기 비율: τ = 0.001 (0.1%)
- 표본 구성: 사기 1,000건 + 정상 9,000건 -> ȳ = 0.1 (10%)
- 보정 항: ln[(0.999/0.001) * (0.1/0.9)] = ln(111.0) ≈ 4.71
- β₀를 4.71만큼 조정
```

## 4. 교정 방법 2: 가중 로지스틱 회귀 (Weighted Logistic Regression)

### 4.1 가중치 설계

각 관측치에 역선택확률 가중치를 부여:

$$w_i = \begin{cases} \frac{\tau}{\bar{y}} & \text{if } y_i = 1 \text{ (양성 사례)} \\ \frac{1-\tau}{1-\bar{y}} & \text{if } y_i = 0 \text{ (음성 사례)} \end{cases}$$

### 4.2 가중 우도 함수

$$\ell_w(\boldsymbol{\beta}) = \sum_{i=1}^n w_i [y_i \ln \pi_i + (1-y_i)\ln(1-\pi_i)]$$

### 4.3 가중치의 직관적 의미

| 상황 | 양성 가중치 | 음성 가중치 | 해석 |
|------|:-----------:|:-----------:|------|
| $\tau = \bar{y}$ | 1.0 | 1.0 | 표본이 인구를 대표 -> 보정 불필요 |
| $\tau < \bar{y}$ | <1.0 | >1.0 | 양성 과대표집 -> 양성 가중치 축소 |
| $\tau > \bar{y}$ | >1.0 | <1.0 | 양성 과소표집 -> 양성 가중치 확대 |

**sklearn에서의 구현**:
```python
from sklearn.linear_model import LogisticRegression

# 클래스 가중치를 통한 보정
model = LogisticRegression(class_weight='balanced')

# 또는 직접 가중치 지정
model = LogisticRegression(class_weight={0: w0, 1: w1})
```

## 5. 교정 방법 3: 편향 보정된 추정 (Bias-Corrected Estimation)

### 5.1 Firth의 벌점화 우도 (Penalized Likelihood)

King과 Zeng는 Firth(1993)의 방법을 적용하여, 편향을 직접 교정하는 방법도 제안한다:

$$\ell^*(\boldsymbol{\beta}) = \ell(\boldsymbol{\beta}) + \frac{1}{2}\log|\mathbf{I}(\boldsymbol{\beta})|$$

여기서 $\mathbf{I}(\boldsymbol{\beta}) = \mathbf{X}^T\mathbf{W}\mathbf{X}$는 피셔 정보 행렬이다.

이 벌점항은:
- MLE를 "축소(shrink)"하여 편향을 줄임
- 분리(separation) 문제(양성/음성이 완벽히 분리되어 MLE가 발산하는 경우)도 해결
- Jeffreys 사전분포를 사용한 베이지안 추정과 동치

### 5.2 보정된 분산 추정

표준 분산 추정량에 대한 유한표본 보정:

$$\text{Var}^*(\hat{\boldsymbol{\beta}}) = (\mathbf{X}^T\hat{\mathbf{W}}\mathbf{X})^{-1} + C$$

여기서 $C$는 유한표본 보정 항이다. 이를 통해 더 정확한 신뢰구간과 p-value를 얻을 수 있다.

## 6. 표본 설계 권고사항

### 6.1 최적 표본 크기

King과 Zeng는 양성 사례 대 음성 사례의 비율에 대한 가이드라인을 제시한다:

- **최소 권고**: 양성 사례의 2~5배의 음성 사례
- **수확 체감**: 음성 사례를 양성의 5배 이상으로 늘려도 효율성 개선이 미미
- **핵심 원칙**: 모든 양성 사례를 포함하고, 음성은 적절히 샘플링

### 6.2 실용적 가이드라인

| 양성 비율 | 권고 전략 |
|:---------:|----------|
| > 20% | 표준 로지스틱 회귀 적용 가능 |
| 5% ~ 20% | 가중 로지스틱 회귀 또는 편향 보정 권고 |
| 1% ~ 5% | 반드시 보정 방법 적용 필요 |
| < 1% | 케이스-통제 설계 + 사전 교정 + 편향 보정 |

## 7. 정치학 응용 사례

### 7.1 국제 분쟁 예측

논문에서 사용한 주요 응용 사례:

**데이터**: 1816-1992년 국가 쌍(dyad) 연도별 관측치
- 전체 관측치: ~300,000
- 분쟁 발생(Y=1): ~3,000 (~1%)
- 설명 변수: 동맹 관계, 경제 상호의존성, 민주주의 수준, 군사력 비율 등

**결과**:
- 표준 로지스틱 회귀의 예측 확률이 실제보다 약 30% 과소추정
- 사전 교정 후 예측 확률이 실제 분쟁 발생률과 더 잘 일치
- 특히 "분쟁 위험이 높은" 국가 쌍에 대한 예측이 크게 개선

## 8. 시뮬레이션 실험 결과

### 8.1 실험 설계

- 다양한 양성 비율(1%, 5%, 10%, 25%, 50%)과 표본 크기(500, 1000, 5000, 10000)에서 시뮬레이션
- 각 조건에서 1,000회 반복 시뮬레이션
- 표준 MLE, 사전 교정, 가중 추정, 편향 보정 비교

### 8.2 주요 결과

1. **표준 MLE**: 양성 비율이 낮을수록 편향이 급격히 증가
2. **사전 교정**: 절편의 편향을 효과적으로 교정하지만, 다른 계수에는 영향 없음
3. **가중 추정**: 전반적으로 편향을 줄이지만, 분산이 약간 증가
4. **편향 보정(Firth)**: 모든 계수에 대해 편향을 효과적으로 줄이며, RMSE(Root Mean Squared Error) 측면에서 가장 우수

## 9. 현대 기계학습과의 연결

King과 Zeng의 논문은 현대 기계학습에서 클래스 불균형(class imbalance) 문제를 다루는 다양한 기법의 통계적 기초를 제공한다:

| King & Zeng (2001) | 현대 기계학습 기법 |
|--------------------|-------------------|
| 가중 로지스틱 회귀 | `class_weight='balanced'` in sklearn |
| 케이스-통제 설계 | 언더샘플링(Undersampling) |
| 모든 양성 포함 + 음성 샘플링 | SMOTE, 오버샘플링(Oversampling) |
| 사전 교정 | 임계값 조정(Threshold moving) |
| 편향 보정 추정 | 벌점화 회귀(Penalized regression) |

이 논문은 이러한 기법들의 **왜 필요한지**에 대한 이론적 근거를 명확히 제시한다는 점에서 기계학습 실무자에게도 매우 유용하다.

## 10. 핵심 수식 총정리

| 개념 | 수식 |
|------|------|
| 표준 로지스틱 모델 | $\pi(x) = \frac{1}{1+e^{-x^T\beta}}$ |
| MLE 편향 | $\text{bias}(\hat{\beta}) = (X^TWX)^{-1}X^TW\xi$ |
| 사전 교정 | $\beta_0^{corr} = \hat{\beta}_0 - \ln\left[\frac{(1-\tau)\bar{y}}{\tau(1-\bar{y})}\right]$ |
| 가중치 (양성) | $w_1 = \tau/\bar{y}$ |
| 가중치 (음성) | $w_0 = (1-\tau)/(1-\bar{y})$ |
| 벌점화 우도 | $\ell^*(\beta) = \ell(\beta) + \frac{1}{2}\log\|I(\beta)\|$ |
