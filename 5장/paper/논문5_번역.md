## 1장: 선형 회귀에서 로지스틱 회귀로

### 1.1 선형 확률 모델 (Linear Probability Model)의 문제

이진 종속 변수 $Y \in \{0, 1\}$에 OLS를 적용하면:

$$\hat{Y} = \hat{\beta}_0 + \hat{\beta}_1 X_1 + \hat{\beta}_2 X_2 + \cdots + \hat{\beta}_k X_k$$

이 모델을 **선형 확률 모델(Linear Probability Model, LPM)**이라 한다. LPM은 다음과 같은 해석을 가진다:

$$E(Y|X) = P(Y=1|X) = \beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k$$

### 1.2 LPM의 네 가지 문제

**문제 1: 확률 범위 위반**

예측 확률 $\hat{P}$가 0 미만이거나 1 초과하는 값을 가질 수 있다. 이는 확률로서 의미가 없다.

수치 예시:
```
나이(X)    실제(Y)   LPM 예측
20         0         1.15    <- 1 초과 (불가능한 확률)
30         0         0.85
50         1         0.25
70         1        -0.05    <- 0 미만 (불가능한 확률)
```

**문제 2: 이분산성 (Heteroscedasticity)**

OLS에서는 오차의 분산이 일정하다고 가정하지만($\text{Var}(\epsilon) = \sigma^2$), 이진 종속 변수에서:

$$\text{Var}(Y|X) = P(Y=1|X) \cdot [1-P(Y=1|X)] = \pi(X)[1-\pi(X)]$$

이 분산은 $\pi(X)$에 의존하므로 이분산적이다.

**문제 3: 비정규 잔차**

$Y$가 0 또는 1만 취하므로, 잔차 $e = Y - \hat{Y}$도 이산적 값만 취한다:
- $Y=1$: $e = 1 - \hat{P}$
- $Y=0$: $e = 0 - \hat{P} = -\hat{P}$

이는 정규분포와 전혀 다르다.

**문제 4: 실제 관계의 비선형성**

확률과 설명 변수 사이의 실제 관계는 S자 곡선(sigmoid curve)을 따르는 경우가 많다. LPM의 직선은 이를 적절히 포착하지 못한다.

### 1.3 로지스틱 회귀의 해법

로지스틱 회귀는 로짓 변환을 통해 이 모든 문제를 해결한다:

$$\text{logit}[\pi(X)] = \ln\left[\frac{\pi(X)}{1-\pi(X)}\right] = \beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k$$

역변환하면:

$$\pi(X) = \frac{e^{\beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k}}{1 + e^{\beta_0 + \beta_1 X_1 + \cdots + \beta_k X_k}}$$

이 변환의 효과:
1. $\pi(X) \in (0, 1)$이 자동으로 보장됨 -> 문제 1 해결
2. MLE는 이분산성을 자동으로 처리 -> 문제 2 해결
3. 베르누이 분포를 명시적으로 가정 -> 문제 3 해결
4. S자 곡선으로 비선형 관계 포착 -> 문제 4 해결

## 2장: 로지스틱 회귀 계수의 해석

### 2.1 로짓 계수 해석 (Logit Coefficient)

가장 기본적인 해석: $\beta_j$는 다른 변수가 일정할 때, $X_j$가 1단위 변할 때의 **로짓 변화량**이다.

$$\Delta \text{logit} = \beta_j \cdot \Delta X_j$$

**해석 예시**: $\beta_{\text{age}} = -0.04$이면
- "나이가 1세 증가할 때, 생존 로짓이 0.04 감소한다"
- OLS의 비표준화 계수에 대응하지만, 단위가 "로짓"이어서 직관적이지 않음

### 2.2 오즈비 해석 (Odds Ratio)

$e^{\beta_j}$로 변환하면 **오즈의 승수적 변화(multiplicative change)**를 나타낸다:

$$\text{OR} = e^{\beta_j}$$

**해석 예시**: $\beta_{\text{age}} = -0.04$이면
- $OR = e^{-0.04} = 0.961$
- "나이가 1세 증가할 때, 생존 오즈가 0.961배가 된다" (3.9% 감소)

오즈비 해석의 장단점:

| 장점 | 단점 |
|------|------|
| 가장 널리 사용됨 | 오즈와 확률을 혼동하기 쉬움 |
| 방향과 크기 동시 파악 | 기준점(baseline)에 따라 확률 변화량이 다름 |
| 통계적 유의성 판단 용이 | OR=1이 "영향 없음"을 의미 |

### 2.3 표준화 로짓 계수 (Standardized Logit Coefficient)

OLS에서의 표준화 계수(beta weight)에 대응:

$$\beta_j^* = \beta_j \cdot \frac{s_{X_j}}{s_{\text{logit}}}$$

여기서:
- $s_{X_j}$: $X_j$의 표준편차
- $s_{\text{logit}}$: 예측된 로짓 값의 표준편차

장점: 서로 다른 단위의 변수 간 **상대적 중요도** 비교가 가능하다.

**해석 예시**:
- $\beta^*_{\text{age}} = -0.15$
- $\beta^*_{\text{sex}} = 0.52$
- -> 성별이 나이보다 생존에 약 3.5배 더 큰 영향을 미침

### 2.4 한계 효과 (Marginal Effect)

확률의 실제 변화량:

$$\frac{\partial \pi}{\partial X_j} = \beta_j \cdot \pi(X) \cdot [1 - \pi(X)]$$

**핵심 특성**: 한계 효과는 현재 확률 수준에 따라 달라진다!

| 현재 확률 $\pi$ | 한계 효과 ($\beta_j = 1$ 가정) |
|:---------------:|:-----------------------------:|
| 0.01 | 0.0099 |
| 0.10 | 0.0900 |
| 0.30 | 0.2100 |
| 0.50 | **0.2500** (최대) |
| 0.70 | 0.2100 |
| 0.90 | 0.0900 |
| 0.99 | 0.0099 |

한계 효과는 $\pi = 0.5$에서 최대이며, 극단값(0 또는 1 근처)에서 거의 0이다. 이것이 시그모이드 함수의 S자 형태를 반영한다.

## 3장: 모델 적합도 평가

### 3.1 유사 R-squared (Pseudo R-squared)

OLS에서의 $R^2$는 로지스틱 회귀에 직접 적용할 수 없다. Menard는 여러 대안 지표를 비교한다:

**McFadden's R^2 (로그 우도 비율 지수)**:

$$R^2_{McF} = 1 - \frac{\ell(\hat{\beta})}{\ell(\hat{\beta}_0)}$$

- $\ell(\hat{\beta})$: 완전 모델의 로그 우도
- $\ell(\hat{\beta}_0)$: 절편만 포함한 모델의 로그 우도
- 범위: [0, 1), 이론적 최대값이 1에 도달하지 않음

**Cox & Snell R^2**:

$$R^2_{CS} = 1 - \left[\frac{L(\hat{\beta}_0)}{L(\hat{\beta})}\right]^{2/n}$$

- 우도비에 기반
- 범위: [0, <1), 최대값이 $1 - L(\hat{\beta}_0)^{2/n}$

**Nagelkerke R^2** (가장 널리 사용):

$$R^2_{N} = \frac{R^2_{CS}}{R^2_{CS,max}} = \frac{1 - [L(\hat{\beta}_0)/L(\hat{\beta})]^{2/n}}{1 - L(\hat{\beta}_0)^{2/n}}$$

- Cox & Snell을 정규화하여 최대값이 1이 되도록 조정
- OLS의 $R^2$와 가장 유사한 해석

**실용적 기준** (Menard의 가이드라인):

| 유사 R^2 값 | 해석 |
|:----------:|------|
| < 0.10 | 낮은 적합도 |
| 0.10 ~ 0.30 | 보통 적합도 |
| 0.30 ~ 0.50 | 좋은 적합도 |
| > 0.50 | 매우 좋은 적합도 |

### 3.2 분류 테이블 분석

다양한 임계값에서의 분류 성능을 체계적으로 분석:

**민감도(Sensitivity)와 특이도(Specificity)의 트레이드오프**:

| 임계값 | 민감도 | 특이도 | 전체 정확도 |
|:------:|:------:|:------:|:----------:|
| 0.30 | 0.90 | 0.60 | 0.72 |
| 0.40 | 0.82 | 0.72 | 0.76 |
| **0.50** | **0.75** | **0.80** | **0.78** |
| 0.60 | 0.65 | 0.88 | 0.79 |
| 0.70 | 0.50 | 0.93 | 0.76 |

Menard는 **연구 목적에 따라 임계값을 조정해야 한다**고 강조한다:
- 질병 스크리닝: 민감도를 우선 (임계값을 낮추어)
- 확정 진단: 특이도를 우선 (임계값을 높여)

### 3.3 ROC 곡선과 AUC

Menard는 ROC 곡선을 "단일 임계값이 아닌, 모든 가능한 임계값에서의 모델 성능을 종합적으로 평가하는 도구"로 소개한다.

AUC(Area Under the Curve)의 해석:
- **0.5**: 무작위 추측 (동전 던지기와 동일)
- **0.7~0.8**: 수용 가능한 판별력
- **0.8~0.9**: 우수한 판별력
- **>0.9**: 뛰어난 판별력

## 4장: 모델 진단 (Model Diagnostics)

### 4.1 다중공선성 (Multicollinearity)

로지스틱 회귀에서 다중공선성이 미치는 영향:
- 계수의 표준오차가 비정상적으로 커짐
- 계수의 부호가 예상과 반대로 나올 수 있음
- 모델의 전반적 적합도는 좋지만 개별 계수가 유의하지 않음

진단 방법:
```
1. VIF (Variance Inflation Factor) 계산
   - OLS 모델에서의 VIF를 로지스틱 회귀에 준용
   - VIF > 10이면 심각한 다중공선성

2. 조건지수 (Condition Index) 확인
   - > 30이면 심각한 다중공선성

3. 계수 안정성 확인
   - 변수 추가/제거 시 계수가 크게 변하면 다중공선성 의심
```

### 4.2 영향력 있는 관측치 (Influential Cases)

특정 관측치가 모델에 과도한 영향을 미치는지 진단:

| 진단 통계량 | 기준 | 의미 |
|------------|:----:|------|
| Studentized 잔차 | \|r\| > 2 | 모델에 잘 적합하지 않는 관측치 |
| 레버리지 | $h > 2(k+1)/n$ | 설명 변수 공간에서 극단적 위치 |
| Cook's D | $D > 4/n$ | 제거 시 모델이 크게 변하는 관측치 |
| DfBeta | \|DfBeta\| > $2/\sqrt{n}$ | 특정 계수에 큰 영향 |

### 4.3 로짓에서의 선형성 (Linearity in the Logit)

로지스틱 회귀의 핵심 가정: 연속형 설명 변수와 로짓 사이의 선형 관계

검증 방법:
1. **Box-Tidwell 검정**: $X_j \ln(X_j)$ 항을 추가하여 유의성 검정
2. **분위수 분석**: $X_j$를 분위수 그룹으로 나누고 각 그룹의 로짓을 플롯
3. **Lowess 스무딩**: 비모수적으로 로짓-설명 변수 관계를 시각화

선형성이 위배될 경우의 해법:
- 변수 변환: $\log(X)$, $X^2$, $\sqrt{X}$ 등
- 구간화(binning): 연속형을 범주형으로 변환
- 스플라인(spline) 사용

## 5장: 다항 및 순서형 로지스틱 회귀

### 5.1 다항 로지스틱 회귀 (Multinomial Logistic Regression)

결과 변수의 범주가 3개 이상이고 순서가 없는 경우:

기준 범주 $K$에 대해:

$$\ln\left[\frac{P(Y=j)}{P(Y=K)}\right] = \beta_{j0} + \beta_{j1}X_1 + \cdots + \beta_{jp}X_p$$

각 비기준 범주에 대해 별도의 로짓 방정식이 존재한다.

### 5.2 순서형 로지스틱 회귀 (Ordinal Logistic Regression)

결과 변수의 범주가 순서를 가지는 경우 (예: 매우불만족/불만족/보통/만족/매우만족):

**누적 로짓 모델 (Cumulative Logit Model)**:

$$\ln\left[\frac{P(Y \leq j)}{P(Y > j)}\right] = \alpha_j - (\beta_1 X_1 + \cdots + \beta_p X_p)$$

핵심 가정: **비례 오즈 가정** - 모든 절단점에서 동일한 $\beta$ 사용

## 6장: 실용적 조언 종합

### 6.1 모델 구축 체크리스트

Menard가 제안하는 실전 체크리스트:

1. **사전 단계**
   - [ ] 이론적 프레임워크에 기반한 변수 선택
   - [ ] 결측치 처리 방법 결정
   - [ ] 카테고리 변수의 더미 코딩 방법 결정

2. **모델 적합 단계**
   - [ ] 전체 모델 적합도 확인 (우도비 검정)
   - [ ] 개별 계수 유의성 확인
   - [ ] 유사 R^2 확인

3. **진단 단계**
   - [ ] 다중공선성 검사 (VIF < 10)
   - [ ] 영향력 있는 관측치 식별
   - [ ] 로짓에서의 선형성 확인
   - [ ] Hosmer-Lemeshow 검정

4. **해석 단계**
   - [ ] 오즈비와 신뢰구간 산출
   - [ ] 분류 테이블 작성
   - [ ] ROC 곡선과 AUC 산출

5. **보고 단계**
   - [ ] 계수, 표준오차, p-value, 오즈비, 95% CI 보고
   - [ ] 적합도 지표 보고
   - [ ] 분류 성능 보고

### 6.2 보고서 작성 가이드

| 보고 항목 | 필수/선택 | 설명 |
|-----------|:--------:|------|
| 표본 크기 | 필수 | 전체 및 각 범주별 |
| 계수와 표준오차 | 필수 | 모든 설명 변수에 대해 |
| 오즈비와 95% CI | 필수 | 해석의 핵심 |
| 우도비 검정 | 필수 | 전체 모델 유의성 |
| 유사 R^2 | 권장 | Nagelkerke 또는 McFadden |
| 분류 테이블 | 권장 | 임계값 명시 |
| ROC / AUC | 선택 | 모델 판별력 |
| H-L 검정 | 선택 | 적합도 검정 |

## 핵심 수식 총정리

| 개념 | 수식 |
|------|------|
| 로짓 모델 | $\text{logit}(\pi) = \beta_0 + \sum \beta_j X_j$ |
| 역변환 (확률) | $\pi = \frac{e^{X\beta}}{1+e^{X\beta}}$ |
| 오즈비 | $OR = e^{\beta_j}$ |
| 한계 효과 | $\partial\pi/\partial X_j = \beta_j \pi(1-\pi)$ |
| 표준화 계수 | $\beta_j^* = \beta_j \cdot s_{X_j}/s_{\text{logit}}$ |
| McFadden R^2 | $R^2_M = 1 - \ell_1/\ell_0$ |
| Nagelkerke R^2 | $R^2_N = R^2_{CS}/R^2_{CS,max}$ |
| 민감도 | $\text{Sensitivity} = TP/(TP+FN)$ |
| 특이도 | $\text{Specificity} = TN/(TN+FP)$ |
