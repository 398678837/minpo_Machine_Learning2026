> **원저**: D. W. Hosmer & S. Lemeshow (2000), "Applied Logistic Regression" (2nd Edition), Wiley-Interscience

## 1장: 로지스틱 회귀 모델의 소개

### 1.1 기본 모델 구조

Hosmer와 Lemeshow는 로지스틱 회귀를 일반화선형모형(GLM)의 관점에서 소개한다. 결과 변수 $Y$가 이진(0 또는 1)일 때, 조건부 평균은:

$$E(Y|x) = P(Y=1|x) = \pi(x)$$

로지스틱 회귀 모델에서 이 조건부 확률은 다음과 같이 모델링된다:

$$\pi(x) = \frac{e^{\beta_0 + \beta_1 x}}{1 + e^{\beta_0 + \beta_1 x}}$$

이를 로짓 변환하면:

$$g(x) = \ln\left[\frac{\pi(x)}{1-\pi(x)}\right] = \beta_0 + \beta_1 x$$

$g(x)$를 **로짓(logit)** 이라 부르며, 이것이 설명 변수의 선형 함수이다.

### 1.2 로지스틱 회귀 vs 선형 회귀의 핵심 차이

저자들은 두 모델의 차이를 다음과 같이 정리한다:

| 측면 | 선형 회귀 | 로지스틱 회귀 |
|------|-----------|--------------|
| $E(Y\|x)$의 범위 | $(-\infty, +\infty)$ | $(0, 1)$ |
| 오차 분포 | 정규분포 | 베르누이 분포 |
| 추정 방법 | 최소제곱법(OLS) | 최대우도추정법(MLE) |
| 분산의 성질 | 등분산(homoscedastic) | 이분산($\pi(x)[1-\pi(x)]$) |

### 1.3 최대우도추정 (MLE) 과정

우도 함수의 구성과 최대화 과정을 단계별로 설명한다:

**1단계: 우도 함수 정의**

$$L(\boldsymbol{\beta}) = \prod_{i=1}^{n} \pi(x_i)^{y_i} [1-\pi(x_i)]^{1-y_i}$$

**2단계: 로그 우도로 변환**

$$\ell(\boldsymbol{\beta}) = \sum_{i=1}^{n} \{y_i \ln[\pi(x_i)] + (1-y_i)\ln[1-\pi(x_i)]\}$$

**3단계: 편미분으로 우도 방정식 도출**

$$\frac{\partial \ell}{\partial \beta_0} = \sum_{i=1}^{n}[y_i - \pi(x_i)] = 0$$

$$\frac{\partial \ell}{\partial \beta_1} = \sum_{i=1}^{n} x_i[y_i - \pi(x_i)] = 0$$

**4단계: 반복적 수치 해법으로 해 구함** (뉴턴-랩슨 또는 IRLS)

## 2장: 유의성 검정과 신뢰구간

### 2.1 우도비 검정 (Likelihood Ratio Test)

완전 모델(full model)과 축소 모델(reduced model)을 비교:

$$G = -2[\ell(\text{reduced}) - \ell(\text{full})]$$

$G$는 점근적으로 $\chi^2$ 분포를 따르며, 자유도는 두 모델의 파라미터 수 차이이다.

**예시**: 설명 변수 $x_1$의 유의성 검정
- 완전 모델: $g(x) = \beta_0 + \beta_1 x_1$
- 축소 모델: $g(x) = \beta_0$ (절편만 있는 모델)
- $G = -2[\ell_0 - \ell_1] \sim \chi^2(1)$

### 2.2 왈드 검정 (Wald Test)

개별 계수의 유의성을 검정하는 보다 간단한 방법:

$$W = \frac{\hat{\beta}_j}{\text{SE}(\hat{\beta}_j)}$$

$W$는 점근적으로 표준정규분포를 따른다. 즉, $W^2 \sim \chi^2(1)$.

**주의**: 왈드 검정은 계수의 절대값이 클 때 검정력이 떨어질 수 있으므로, 우도비 검정이 일반적으로 더 권장된다.

### 2.3 계수의 신뢰구간

$\beta_j$의 100(1-$\alpha$)% 왈드 신뢰구간:

$$\hat{\beta}_j \pm z_{1-\alpha/2} \cdot \text{SE}(\hat{\beta}_j)$$

오즈비의 신뢰구간:

$$e^{\hat{\beta}_j \pm z_{1-\alpha/2} \cdot \text{SE}(\hat{\beta}_j)}$$

## 3장: 계수 해석 (Interpretation of Coefficients)

### 3.1 이진 설명 변수의 해석

설명 변수 $x$가 0 또는 1인 경우:

$$\text{Odds Ratio} = e^{\hat{\beta}_1} = \frac{\text{Odds}(x=1)}{\text{Odds}(x=0)}$$

**타이타닉 예시**: 성별(Sex_male)의 계수가 -2.569이면:
- $OR = e^{-2.569} = 0.077$
- 해석: "남성의 생존 오즈가 여성에 비해 약 0.077배 (즉, 92.3% 낮음)"

### 3.2 연속형 설명 변수의 해석

$x$가 $c$단위 변화할 때의 오즈비:

$$OR(c) = e^{c \cdot \hat{\beta}_1}$$

1단위 변화일 때: $OR = e^{\hat{\beta}_1}$

**타이타닉 예시**: Age의 계수가 -0.040이면:
- 1세 증가 시: $OR = e^{-0.040} = 0.961$ (오즈 3.9% 감소)
- 10세 증가 시: $OR = e^{-0.40} = 0.670$ (오즈 33% 감소)

### 3.3 다변량 모델에서의 해석: 조정된 오즈비

다변량 모델에서 각 계수의 오즈비는 **다른 변수들을 통제(조정)한 상태에서의** 효과를 나타낸다:

$$\text{Adjusted OR}(x_j) = e^{\hat{\beta}_j}$$

"다른 모든 변수가 일정할 때, $x_j$가 1단위 증가하면 오즈가 $e^{\hat{\beta}_j}$배 변한다."

## 4장: 모델 구축 전략 (Model-Building Strategies)

### 4.1 Purposeful Selection 방법론

Hosmer와 Lemeshow가 제안한 체계적 변수 선택 절차:

**단계 1: 단변량 분석**
- 각 설명 변수를 개별적으로 결과 변수와의 관련성을 분석
- 유의 수준 기준: p < 0.25 (보수적 기준)
- p < 0.25인 변수들을 다변량 모델의 후보로 선정

**단계 2: 다변량 모델 적합**
- 단계 1에서 선택된 모든 변수를 포함한 모델 적합
- 각 변수의 p-value 확인

**단계 3: 비유의한 변수 제거**
- p > 0.05인 변수를 하나씩 제거 (가장 큰 p-value부터)
- 매 제거 후 남은 변수의 계수 변화 확인

**단계 4: 교란변수(confounder) 확인**
- 변수 제거 후 남은 계수가 20% 이상 변화하면, 제거된 변수는 교란변수
- 교란변수는 p-value가 유의하지 않아도 모델에 유지

**단계 5: 연속형 변수의 로짓에서의 선형성 확인**
- Lowess 스무딩, 분위수(quartile) 기반 분석 등 활용
- 선형성이 위배되면 적절한 변환(제곱, 로그 등) 적용

**단계 6: 교호작용(interaction) 탐색**
- 이론적으로 또는 데이터에서 의미 있는 교호작용 항 추가
- 통계적 유의성과 실질적 의미를 모두 고려

**단계 7: 최종 모델 평가**
- Hosmer-Lemeshow 검정, ROC 분석 등으로 적합도 평가

### 4.2 단계적 선택법 (Stepwise Selection)

자동화된 변수 선택 방법:

| 방법 | 설명 |
|------|------|
| 전진 선택(Forward) | 절편만 있는 모델에서 시작, 유의한 변수를 하나씩 추가 |
| 후진 제거(Backward) | 모든 변수를 포함한 모델에서 시작, 비유의한 변수를 하나씩 제거 |
| 단계적(Stepwise) | 전진+후진을 결합, 매 단계에서 추가/제거 동시 고려 |

**주의사항**: 자동화된 방법은 교란변수, 교호작용, 비선형성 등을 적절히 처리하지 못할 수 있으므로, purposeful selection이 일반적으로 더 권장된다.

## 5장: 적합도 평가 (Goodness-of-Fit Assessment)

### 5.1 Hosmer-Lemeshow 검정

가장 널리 사용되는 로지스틱 회귀 적합도 검정:

**절차**:
1. 예측 확률 $\hat{\pi}_i$를 기준으로 관측치를 오름차순 정렬
2. $g$개의 동일 크기 그룹으로 분할 (보통 $g=10$, 즉 십분위수)
3. 각 그룹 $k$에서:
   - 관측된 사건 수: $O_k = \sum_{i \in \text{group}_k} y_i$
   - 기대 사건 수: $E_k = \sum_{i \in \text{group}_k} \hat{\pi}_i$
4. 검정 통계량 계산:

$$\hat{C} = \sum_{k=1}^{g} \frac{(O_k - E_k)^2}{n_k \bar{\pi}_k (1-\bar{\pi}_k)}$$

5. $\hat{C} \sim \chi^2(g-2)$ 하에서 p-value 계산

**해석**:
- p-value > 0.05: 모델이 데이터를 잘 설명함 (귀무가설 기각 실패)
- p-value < 0.05: 모델 적합도에 문제가 있을 수 있음

### 5.2 분류 테이블 (Classification Table)

다양한 임계값(cutoff)에서의 분류 성능:

| 지표 | 공식 | 의미 |
|------|------|------|
| 민감도 (Sensitivity) | $TP/(TP+FN)$ | 실제 양성 중 양성으로 예측한 비율 |
| 특이도 (Specificity) | $TN/(TN+FP)$ | 실제 음성 중 음성으로 예측한 비율 |
| 양성 예측도 (PPV) | $TP/(TP+FP)$ | 양성 예측 중 실제 양성 비율 |
| 음성 예측도 (NPV) | $TN/(TN+FN)$ | 음성 예측 중 실제 음성 비율 |
| 정확도 (Accuracy) | $(TP+TN)/N$ | 전체 올바른 분류 비율 |

### 5.3 ROC 곡선 분석

ROC(Receiver Operating Characteristic) 곡선은 모든 가능한 임계값에서의 민감도와 1-특이도를 그린 곡선이다.

- **x축**: 1-Specificity (위양성률, False Positive Rate)
- **y축**: Sensitivity (진양성률, True Positive Rate)
- **AUC (Area Under the Curve)**: 모델의 전반적 판별력

$$\text{AUC} = P(\hat{\pi}_{Y=1} > \hat{\pi}_{Y=0})$$

즉, AUC는 무작위로 선택한 양성 사례의 예측 확률이 무작위로 선택한 음성 사례의 예측 확률보다 클 확률이다.

## 6장: 다항 로지스틱 회귀 (Multinomial Logistic Regression)

### 6.1 3개 이상의 범주가 있는 경우

결과 변수가 $K$개의 범주를 가질 때, 기준 범주(reference category)를 $K$로 설정하면:

$$\ln\left[\frac{P(Y=j|x)}{P(Y=K|x)}\right] = \beta_{j0} + \beta_{j1}x_1 + \cdots + \beta_{jp}x_p$$

$j = 1, 2, \ldots, K-1$에 대해 $(K-1)$개의 로짓 방정식이 존재한다.

각 범주의 확률:

$$P(Y=j|x) = \frac{e^{g_j(x)}}{1 + \sum_{k=1}^{K-1} e^{g_k(x)}}, \quad j = 1, \ldots, K-1$$

$$P(Y=K|x) = \frac{1}{1 + \sum_{k=1}^{K-1} e^{g_k(x)}}$$

이것이 바로 **소프트맥스 함수(softmax function)**의 일반적 형태이다.

## 7장: 순서형 로지스틱 회귀 (Ordinal Logistic Regression)

### 7.1 비례 오즈 모델 (Proportional Odds Model)

결과 변수의 범주가 순서를 가질 때 (예: 경증/중등증/중증):

$$\ln\left[\frac{P(Y \leq j|x)}{P(Y > j|x)}\right] = \alpha_j - \boldsymbol{\beta}^T \mathbf{x}$$

핵심 가정: **비례 오즈 가정(proportional odds assumption)** - 각 절단점(cutpoint)에서의 오즈비가 동일하다.

## 핵심 통찰 요약

이 교재가 기계학습 실무자에게 제공하는 핵심 교훈:

1. **모델 구축은 반복적 과정**: 한 번에 완벽한 모델을 만들 수 없으며, 구축-평가-수정의 순환을 거쳐야 한다
2. **정확도만으로는 부족**: 적합도 검정, ROC 분석, 잔차 진단 등 다각적 평가가 필요하다
3. **해석 가능성의 가치**: 로지스틱 회귀의 최대 장점은 오즈비를 통한 직관적 해석이다
4. **도메인 지식의 중요성**: 통계적 유의성만으로 변수를 선택하면 교란변수를 놓칠 수 있다
5. **임계값은 맥락에 따라 결정**: 0.5가 항상 최적이 아니며, 비즈니스/의학적 맥락에 따라 조정해야 한다
