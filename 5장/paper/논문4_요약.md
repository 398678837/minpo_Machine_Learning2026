## 논문 개요 (300+ words)

Andrew Ng와 Michael Jordan의 2002년 논문 "On Discriminative vs. Generative classifiers"는 기계학습의 두 가지 근본적인 접근법 - 판별 모델(discriminative model)과 생성 모델(generative model) - 을 이론적, 실험적으로 비교한 획기적인 논문이다. 이 논문은 로지스틱 회귀(discriminative)와 나이브 베이즈(generative)를 대표 사례로 사용하여, 두 접근법의 근본적 차이와 각각의 장단점을 명쾌하게 분석하였다.

판별 모델은 입력 $\mathbf{x}$가 주어졌을 때 레이블 $y$의 조건부 분포 $P(y|\mathbf{x})$를 직접 학습한다. 반면 생성 모델은 결합 분포 $P(\mathbf{x}, y)$ 또는 동등하게 $P(\mathbf{x}|y)$와 $P(y)$를 학습한 후, 베이즈 정리를 통해 $P(y|\mathbf{x})$를 유도한다.

논문의 가장 중요한 이론적 기여는 다음과 같다. 첫째, 나이브 베이즈(생성 모델)와 로지스틱 회귀(판별 모델)가 동일한 모델 패밀리(model family)를 공유함을 보였다. 구체적으로, 나이브 베이즈의 가정(특성 독립성, 각 특성의 조건부 분포가 지수족)이 성립할 때, 최적의 분류기는 로지스틱 회귀의 형태를 가진다. 그러나 두 모델의 파라미터 추정 방법이 다르기 때문에 다른 결과를 산출한다.

둘째, 두 접근법의 **표본 효율성(sample efficiency)** 에서 근본적 차이가 존재함을 증명하였다:
- **나이브 베이즈**: 점근적 오차(asymptotic error)에 더 빠르게 수렴. $O(\log n)$ 표본에서 점근적 성능에 근접.
- **로지스틱 회귀**: 점근적 오차 자체가 더 낮음(더 정확). 그러나 수렴 속도는 $O(n)$으로 더 느림.

셋째, 이로부터 실용적으로 매우 중요한 결론이 도출된다: **표본이 작을 때는 나이브 베이즈(생성 모델)가 우수하고, 표본이 클 때는 로지스틱 회귀(판별 모델)가 우수하다.** 그리고 이 "전환점(crossover point)"이 존재하여, 특정 표본 크기 이후부터 판별 모델이 생성 모델을 능가하기 시작한다.

이 논문은 판별 모델과 생성 모델의 선택이 단순한 취향의 문제가 아니라, 데이터의 양과 모델 가정의 정확성에 따라 체계적으로 결정해야 하는 설계 결정임을 보여준 중요한 연구이다.

## 핵심 기여 (Key Contributions)

1. **판별-생성 쌍의 이론적 관계 규명**: 나이브 베이즈와 로지스틱 회귀가 동일한 모델 패밀리를 공유하지만 다른 파라미터 추정 방법을 사용함을 증명
2. **수렴 속도의 차이 증명**: 생성 모델의 $O(\log n)$ 수렴 vs 판별 모델의 $O(n)$ 수렴
3. **점근적 성능의 차이**: 판별 모델의 점근적 오차가 생성 모델보다 낮거나 같음을 증명
4. **전환점(Crossover)의 존재**: 표본 크기에 따라 최적 모델이 바뀌는 전환점의 존재를 이론적, 실험적으로 확인
5. **실용적 가이드라인**: 데이터 양에 따른 모델 선택 전략 제시

## 핵심 개념 (Core Concepts)

### 1. 판별 모델 (Discriminative Model)
- **학습 대상**: $P(y|\mathbf{x})$ (조건부 분포)를 직접 학습
- **대표 모델**: 로지스틱 회귀, SVM, 신경망
- **장점**: 결정 경계(decision boundary)를 직접 학습하므로 점근적으로 더 정확
- **단점**: 더 많은 데이터가 필요

### 2. 생성 모델 (Generative Model)
- **학습 대상**: $P(\mathbf{x}|y)$와 $P(y)$를 학습한 후 베이즈 정리로 $P(y|\mathbf{x})$ 유도
- **대표 모델**: 나이브 베이즈, 가우시안 혼합 모델(GMM), HMM
- **장점**: 적은 데이터에서도 효과적, 모델 가정이 맞으면 매우 효율적
- **단점**: 모델 가정이 틀리면 점근적으로도 편향 존재

### 3. 나이브 베이즈와 로지스틱 회귀의 관계

나이브 베이즈 가정 하에서:

$$P(y=1|\mathbf{x}) = \frac{P(\mathbf{x}|y=1)P(y=1)}{P(\mathbf{x})}$$

특성 독립성과 지수족 가정이 성립하면:

$$\log\frac{P(y=1|\mathbf{x})}{P(y=0|\mathbf{x})} = \theta_0 + \sum_j \theta_j x_j$$

이것이 바로 **로지스틱 회귀의 형태**이다!

즉, 나이브 베이즈의 가정이 정확히 성립하는 세계에서, 최적 분류기는 로지스틱 회귀의 형태를 가진다. 차이는 파라미터를 추정하는 방법에 있다.

### 4. 수렴 속도와 점근적 성능의 트레이드오프

| 특성 | 나이브 베이즈 (생성) | 로지스틱 회귀 (판별) |
|------|:--------------------:|:--------------------:|
| 점근적 오차 | 높거나 같음 | 낮거나 같음 |
| 수렴 속도 | $O(\log n)$ (빠름) | $O(n)$ (느림) |
| 소표본 성능 | 우수 | 열등 |
| 대표본 성능 | 열등 | 우수 |

이 논문은 5장에서 배우는 로지스틱 회귀의 **위치와 특성**을 기계학습의 큰 그림 속에서 이해하는 데 핵심적이다.

| 5장 내용 | Ng & Jordan과의 연결 |
|----------|---------------------|
| 로지스틱 회귀의 특성 | 판별 모델의 대표 사례로서의 로지스틱 회귀의 이론적 위치를 명확히 함 |
| 나이브 베이즈(7장)와의 관계 | 두 모델이 동일한 모델 패밀리를 공유하지만 추정 방법이 다름을 설명 |
| 모델 선택 기준 | 데이터 양에 따라 모델을 선택해야 한다는 실용적 기준 제시 |
