## 논문 개요 (300+ words)

Hosmer와 Lemeshow의 "Applied Logistic Regression"은 로지스틱 회귀의 실용적 적용에 관한 가장 권위 있는 교과서 중 하나이다. 2000년에 출판된 2판은 로지스틱 회귀 모델의 구축, 해석, 진단, 그리고 평가에 이르는 전 과정을 체계적으로 다루며, 특히 의학 및 공중보건(public health) 분야에서의 실전 응용에 초점을 맞추고 있다.

이 교재는 단순히 로지스틱 회귀의 수학적 이론을 소개하는 데 그치지 않고, 실제 데이터를 사용하여 모델을 구축하는 전 과정을 단계별로 안내한다. 저자들은 "모델 구축은 예술이자 과학"이라는 철학 하에, 변수 선택(variable selection), 모델 진단(model diagnostics), 적합도 평가(goodness-of-fit assessment) 등 실무에서 반드시 필요한 주제들을 깊이 있게 다룬다.

책의 구조는 크게 세 부분으로 나뉜다. 첫째, 이진 로지스틱 회귀의 기초 이론과 파라미터 추정 방법을 다룬다. 둘째, 모델 구축의 실용적 전략 - 유의미한 변수를 식별하고, 교호작용과 교란변수를 처리하며, 모델의 적합도를 평가하는 방법 - 을 제시한다. 셋째, 다항 로지스틱 회귀(multinomial logistic regression)와 순서형 로지스틱 회귀(ordinal logistic regression) 등 확장 모델을 소개한다.

특히 Hosmer-Lemeshow 검정(Hosmer-Lemeshow goodness-of-fit test)은 이 책에서 처음 체계적으로 소개된 적합도 검정 방법으로, 로지스틱 회귀 모델이 데이터를 얼마나 잘 설명하는지를 평가하는 표준 도구로 자리 잡았다. 이 검정은 예측 확률을 기준으로 데이터를 그룹으로 나눈 후, 관측 빈도와 기대 빈도를 비교하여 모델의 적합도를 통계적으로 검정한다.

실무적 관점에서 이 교재의 가장 큰 가치는 "목적에 부합하는 모델을 어떻게 구축할 것인가"에 대한 체계적인 프레임워크를 제공한다는 점이다. 예측(prediction)이 목적인지, 인과관계 추론(inference)이 목적인지에 따라 모델 구축 전략이 달라져야 한다는 점을 명확히 하고, 각 목적에 맞는 접근법을 제시한다.

## 핵심 기여 (Key Contributions)

1. **체계적 모델 구축 프레임워크**: 변수 선택 -> 모델 적합 -> 진단 -> 평가의 순환적 과정을 체계화
2. **Hosmer-Lemeshow 적합도 검정**: 로지스틱 회귀 모델의 적합도를 평가하는 통계적 검정 방법 제시
3. **모델 진단 도구**: 영향력 있는 관측치(influential observations) 식별, 잔차 분석 등 진단 기법 상세 소개
4. **실전 사례 기반 교육**: Low Birth Weight, ICU, GLOW 등 실제 의학 데이터를 통한 실습 중심 교육
5. **다항/순서형 확장**: 이진 로지스틱 회귀를 넘어 다중 클래스 문제로의 확장 방법론 제시

## 핵심 개념 (Core Concepts)

### 1. 목적 변수의 코딩 (Coding of Outcome Variable)
- 이진 결과: 0 (사건 미발생) / 1 (사건 발생)
- 코딩 방식에 따라 계수의 해석이 달라짐
- 일관된 코딩의 중요성 강조

### 2. Purposeful Selection of Variables (목적적 변수 선택)
Hosmer와 Lemeshow가 제안한 변수 선택의 7단계 과정:

1. **단변량 분석**: 각 변수를 개별적으로 결과 변수와의 관계 분석 (p < 0.25 기준)
2. **다변량 모델 적합**: 유의한 변수들을 포함한 모델 적합
3. **유의하지 않은 변수 제거**: p > 0.05인 변수 순차적 제거
4. **교란변수 확인**: 제거된 변수가 남은 계수에 미치는 영향 확인 (20% 이상 변화 시 재포함)
5. **연속형 변수의 선형성 확인**: 로짓에서의 선형성(linearity in the logit) 가정 검토
6. **교호작용 추가**: 의미 있는 교호작용 항 탐색 및 추가
7. **최종 모델 적합도 평가**: Hosmer-Lemeshow 검정 등으로 평가

### 3. Hosmer-Lemeshow Goodness-of-Fit Test
- 예측 확률을 기준으로 데이터를 g개 그룹(보통 10개)으로 분할
- 각 그룹에서 관측된 사건 수와 기대 사건 수를 비교
- 검정 통계량: $\hat{C} = \sum_{k=1}^{g} \frac{(O_k - E_k)^2}{E_k(1 - E_k/n_k)}$
- 귀무가설: "모델이 데이터를 잘 설명한다"
- p-value가 높으면 모델 적합도가 좋다고 판단

### 4. 진단 통계량 (Diagnostic Statistics)
| 통계량 | 목적 |
|--------|------|
| 피어슨 잔차 (Pearson residual) | 개별 관측치의 적합도 확인 |
| 이탈도 잔차 (Deviance residual) | 각 관측치가 전체 이탈도에 기여하는 정도 |
| 레버리지 (Leverage) | 설계 행렬 공간에서 관측치의 위치 |
| dfbeta | 특정 관측치 제거 시 계수 변화량 |
| Cook의 거리 (Cook's distance) | 영향력의 종합적 측도 |

### 5. ROC 곡선과 c-통계량
- ROC 곡선: 다양한 임계값에서의 민감도(sensitivity)와 1-특이도(1-specificity)를 그린 곡선
- c-통계량(concordance statistic) = AUC (Area Under the ROC Curve)
- c = 0.5: 무작위 추측 수준
- c = 0.7~0.8: 수용 가능한 판별력
- c = 0.8~0.9: 우수한 판별력
- c > 0.9: 뛰어난 판별력

## 5장(로지스틱 회귀)과의 관련성

이 교재는 5장에서 다루는 로지스틱 회귀의 **실용적 적용 전반**에 대한 깊이 있는 참고 자료이다.

| 5장 내용 | Hosmer & Lemeshow과의 연결 |
|----------|---------------------------|
| 시그모이드 함수 / 로그 손실 | Chapter 1에서 로지스틱 회귀의 수학적 기초 상세 설명 |
| 오즈비 해석 | Chapter 3에서 계수 해석과 오즈비의 실용적 의미를 사례와 함께 설명 |
| 전처리 (변수 제거, 더미 변수) | Chapter 4에서 목적적 변수 선택 방법론 제시 |
| 피처 엔지니어링 | Chapter 4에서 교호작용 항 추가, 변수 변환 등 다룸 |
| 모델 평가 (accuracy_score) | Chapter 5에서 적합도 평가, ROC 분석, 분류 테이블 등 포괄적 평가 방법 제시 |
| Confusion Matrix | Chapter 5에서 분류 테이블의 민감도, 특이도, 정확도 해석 상세 설명 |

특히 5장에서 `accuracy_score`만으로 모델을 평가하는 것을 넘어서, Hosmer-Lemeshow 검정, ROC 분석, 잔차 진단 등 다층적 평가의 필요성을 이해하는 데 이 교재가 큰 도움이 된다.
