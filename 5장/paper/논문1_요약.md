## 논문 개요 (300+ words)

D. R. Cox의 1958년 논문 "The Regression Analysis of Binary Sequences"는 로지스틱 회귀(Logistic Regression)의 이론적 토대를 정립한 기념비적 논문이다. 이 논문은 이진(binary) 결과 변수에 대한 회귀 분석 방법론을 최초로 체계적으로 정리하였으며, 현대 통계학과 기계학습에서 가장 널리 사용되는 분류 모델의 원형을 제시하였다.

Cox는 이 논문에서 종속 변수가 0 또는 1의 값만 취하는 이진 데이터(binary data)에 대해, 설명 변수(explanatory variables)와 반응 확률(response probability) 사이의 관계를 모델링하는 문제를 다룬다. 일반적인 선형 회귀(linear regression)가 연속적인 종속 변수를 다루는 반면, 이진 결과 변수에 대해서는 확률값이 0과 1 사이에 놓여야 한다는 제약이 존재한다. Cox는 이 문제를 해결하기 위해 로짓 변환(logit transformation)을 도입하였다.

논문의 핵심 아이디어는 사건 발생 확률 p와 설명 변수 x 사이의 관계를 다음과 같이 설정하는 것이다:

$$\log\left(\frac{p}{1-p}\right) = \beta_0 + \beta_1 x_1 + \beta_2 x_2 + \cdots + \beta_k x_k$$

이 로짓 모델(logit model)은 확률을 (-무한대, +무한대) 범위의 선형 결합으로 매핑하여, 선형 모델의 장점을 유지하면서도 확률의 제약 조건을 자연스럽게 만족시킨다. Cox는 이 모델의 파라미터 추정을 위해 최대우도추정법(Maximum Likelihood Estimation, MLE)을 사용할 것을 제안하였다.

또한 Cox는 모델의 적합도 검정(goodness-of-fit test), 파라미터의 유의성 검정, 그리고 예측 정확도 평가에 대한 방법론도 함께 제시하였다. 특히 최대우도추정량의 점근적 성질(asymptotic properties)을 활용하여, 대표본(large sample)에서의 추론 절차를 체계화하였다.

이 논문은 단순히 이론적 기여에 그치지 않고, 의학 연구(투약량-반응 관계), 생물학(독성 실험), 사회과학(투표 행동 분석) 등 다양한 분야에서의 응용 가능성을 보여주었다. Cox의 이 논문 이후 로지스틱 회귀는 일반화선형모형(Generalized Linear Models, GLM)의 특수한 경우로 정립되었으며, 현대 기계학습에서 분류 문제의 기본 모델로 자리 잡게 되었다.

## 핵심 기여 (Key Contributions)

1. **로짓 변환의 체계적 도입**: 이진 반응 변수에 대해 로짓 변환(log-odds transformation)을 적용하여 선형 모델로 확률을 추정하는 방법론을 정립
2. **최대우도추정법(MLE)의 적용**: 로지스틱 회귀 파라미터 추정에 MLE를 사용하는 방법과 그 점근적 성질을 체계화
3. **모델 진단 및 적합도 검정**: 적합도 검정(goodness-of-fit), 우도비 검정(likelihood ratio test) 등 모델 평가 프레임워크 제시
4. **프로빗 모델과의 비교**: 기존의 프로빗(probit) 모델과 로짓(logit) 모델을 비교하여, 로짓 모델의 해석적 장점(오즈비 해석)을 강조
5. **다중 설명 변수로의 확장**: 단일 설명 변수에서 다중 설명 변수로의 자연스러운 확장 가능성을 논의

## 핵심 개념 (Core Concepts)

### 이진 시퀀스 (Binary Sequences)
- 관측값이 0 또는 1 (성공/실패, 생존/사망 등)로만 구성되는 데이터
- 각 관측치는 독립적인 베르누이 시행(Bernoulli trial)으로 가정

### 로짓 변환 (Logit Transformation)
- 확률 p를 log(p/(1-p))로 변환
- 이 변환을 통해 (0,1) 범위의 확률을 (-무한대, +무한대) 범위의 실수로 매핑
- 시그모이드 함수의 역함수

### 최대우도추정법 (Maximum Likelihood Estimation)
- 관측된 데이터의 우도(likelihood)를 최대화하는 파라미터를 찾는 방법
- 로지스틱 회귀에서는 닫힌 형태(closed-form)의 해가 존재하지 않아 반복적 수치 방법(iterative numerical methods) 사용

### 프로빗 모델과의 비교
- 프로빗(probit): 정규분포의 누적분포함수를 링크 함수로 사용
- 로짓(logit): 로지스틱 분포의 누적분포함수를 사용
- 두 모델은 실용적으로 매우 유사한 결과를 제공하나, 로짓 모델이 오즈비(odds ratio) 해석에서 더 직관적

## 5장(로지스틱 회귀)과의 관련성

이 논문은 5장에서 다루는 로지스틱 회귀의 **수학적 기초 전체**를 정립한 원천 논문이다.

| 5장 내용 | 논문과의 연결 |
|----------|--------------|
| 시그모이드 함수 | Cox가 제안한 로짓 변환의 역함수가 바로 시그모이드 함수 |
| 로그 손실 (Log Loss) | MLE에서 음의 로그우도(negative log-likelihood)를 최소화하는 것이 로그 손실 최소화와 동일 |
| 오즈비(Odds Ratio) 해석 | Cox가 로짓 모델의 핵심 장점으로 강조한 해석 방법 |
| 계수(coefficient) 해석 | 회귀 계수의 지수 변환(exp)이 오즈비가 되는 성질 |
| 경사하강법을 통한 학습 | MLE의 반복적 수치 해법이 현대의 경사하강법으로 발전 |

Cox의 이 1958년 논문을 이해하면, 5장에서 배우는 로지스틱 회귀의 모든 수학적 기초 - 왜 시그모이드 함수를 사용하는지, 왜 로그 손실을 비용 함수로 사용하는지, 왜 오즈비로 계수를 해석하는지 - 를 근본적으로 이해할 수 있다.
