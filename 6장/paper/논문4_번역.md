# 논문 4 상세 번역 및 해설: Distance Metric Learning for Large Margin Nearest Neighbor Classification

## 서지 정보
- **저자**: K. Q. Weinberger, L. K. Saul
- **제목**: Distance Metric Learning for Large Margin Nearest Neighbor Classification
- **학술지**: JMLR, 2009

## 1. 문제 인식: 왜 거리 메트릭을 학습해야 하는가?

### 1.1 KNN의 근본적 한계

KNN 분류기의 성능은 사용하는 **거리 메트릭(distance metric)**에 결정적으로 의존한다. 표준 유클리드 거리는:

$$d_{Euc}(x_i, x_j) = \sqrt{\sum_{k=1}^{d} (x_{ik} - x_{jk})^2}$$

이 거리는 모든 특성에 동일한 가중치를 부여하고, 특성 간의 상관관계를 무시한다. 이는 다음과 같은 문제를 야기한다:

1. **불필요한 특성의 영향**: 분류에 무관한 특성도 거리 계산에 기여하여 노이즈를 추가
2. **특성 간 상관관계 무시**: 상관된 특성들의 공동 패턴을 포착하지 못함
3. **스케일 의존성**: 특성의 스케일에 따라 결과가 달라짐 (스케일링으로 부분적 해결 가능)

### 1.2 마할라노비스 거리의 일반성

마할라노비스 거리는 유클리드 거리의 자연스러운 일반화이다:

$$d_M(x_i, x_j) = \sqrt{(x_i - x_j)^T M (x_i - x_j)}$$

M = L^T L로 분해하면 (L은 d' x d 행렬):

$$d_M(x_i, x_j) = \|L x_i - L x_j\|_2$$

즉, 마할라노비스 거리 학습은 **입력 공간을 선형 변환한 새로운 공간에서의 유클리드 거리**를 학습하는 것과 동일하다. L은 d차원에서 d'차원으로의 투영(projection)으로 해석할 수 있으며, d' < d이면 차원 축소의 효과도 있다.

## 2. LMNN 알고리즘 상세

### 2.1 타깃 이웃 (Target Neighbors)

각 학습 데이터 포인트 x_i에 대해, **같은 클래스에 속하면서 유클리드 거리가 가장 가까운 K개의 데이터 포인트**를 타깃 이웃으로 선정한다.

타깃 이웃 관계를 이진 행렬 eta_ij로 표현한다:
- eta_ij = 1: x_j가 x_i의 타깃 이웃
- eta_ij = 0: 그 외

### 2.2 침입자 (Imposters)

침입자(imposter)란, **다른 클래스에 속하면서 타깃 이웃보다 x_i에 가까운 데이터 포인트**를 말한다. 침입자의 존재는 KNN 분류 오류를 야기한다.

x_l이 x_i의 침입자가 되는 조건:
- y_i != y_l (다른 클래스)
- d_M(x_i, x_l) <= d_M(x_i, x_j) + 1 (타깃 이웃보다 가깝거나 마진 1 이내)

### 2.3 목적 함수

LMNN의 목적 함수는 두 가지 손실의 가중합이다:

$$\mathcal{L}(M) = (1-\mu) \epsilon_{pull}(M) + \mu \epsilon_{push}(M)$$

**당기기 손실 (Pull Loss)**:
$$\epsilon_{pull}(M) = \sum_{i} \sum_{j: \eta_{ij}=1} d_M^2(x_i, x_j)$$

같은 클래스의 타깃 이웃들을 가까이 당긴다.

**밀치기 손실 (Push Loss)**:
$$\epsilon_{push}(M) = \sum_{i} \sum_{j: \eta_{ij}=1} \sum_{l: y_l \neq y_i} \left[1 + d_M^2(x_i, x_j) - d_M^2(x_i, x_l)\right]_+$$

여기서 [z]_+ = max(0, z)는 힌지 손실(hinge loss)이다.

이 손실은 다른 클래스의 포인트 x_l이 타깃 이웃 x_j보다 x_i에 가까울 때 (마진 1 이내) 벌점을 부여한다.

### 2.4 최적화

M에 대해 위 목적 함수를 최소화한다. 제약 조건:
- M >= 0 (양의 반정치)
- xi_ijl >= 0 (슬랙 변수)

이 문제는 **볼록 문제(convex problem)**이므로, 전역 최적해가 보장된다.

실용적으로는 다음과 같은 최적화 전략이 사용된다:
1. **서브그래디언트 하강법(Subgradient Descent)**: 가장 단순하지만 수렴이 느림
2. **투영된 경사 하강법(Projected Gradient Descent)**: M의 양의 반정치 제약을 투영으로 유지
3. **활성 집합 방법(Active Set Method)**: 침입자를 동적으로 관리하여 효율성 향상

## 3. 그래디언트 계산

### 3.1 당기기 손실의 그래디언트

$$\frac{\partial \epsilon_{pull}}{\partial M} = \sum_{i} \sum_{j: \eta_{ij}=1} (x_i - x_j)(x_i - x_j)^T$$

이는 타깃 이웃 쌍의 차이 벡터의 외적(outer product)의 합이다.

### 3.2 밀치기 손실의 그래디언트

활성 제약(active constraint, 힌지 손실이 0보다 큰 경우)에 대해:

$$\frac{\partial \epsilon_{push}}{\partial M} = \sum_{(i,j,l) \in \text{active}} \left[(x_i - x_j)(x_i - x_j)^T - (x_i - x_l)(x_i - x_l)^T\right]$$

---

## 4. 선형 변환으로서의 해석

### 4.1 M = L^T L 분해

M이 양의 반정치이므로, M = L^T L로 분해할 수 있다 (Cholesky 분해 또는 고유값 분해).

L은 d차원 입력을 d'차원으로 매핑하는 선형 변환이다:

$$z_i = L x_i$$

변환된 공간에서의 유클리드 거리가 원래 공간에서의 마할라노비스 거리와 동일하다:

$$d_M(x_i, x_j) = \|z_i - z_j\|_2$$

### 4.2 차원 축소로서의 해석

M의 고유값 분해를 통해 0에 가까운 고유값에 대응하는 차원을 제거하면, 자연스러운 **감독 차원 축소(supervised dimensionality reduction)**가 가능하다. 이는 PCA가 비감독(unsupervised)인 것과 대비되는 중요한 특성이다.

## 5. 실험 결과

### 5.1 MNIST 숫자 인식

| 방법 | 1-NN 오류율 |
|------|-----------|
| 유클리드 KNN | 2.97% |
| PCA + KNN | 2.42% |
| LMNN | 1.20% |
| SVM (RBF) | 1.40% |

LMNN이 유클리드 KNN 대비 오류율을 60% 이상 감소시키며, SVM보다도 우수한 성능을 보인다.

### 5.2 UCI 벤치마크

Iris, Wine, Digits 등 다양한 UCI 데이터셋에서 일관되게 유클리드 KNN을 능가한다.

## 6. 다중 스케일 LMNN (Multi-Scale LMNN)

### 6.1 동기

전역 마할라노비스 거리 하나만으로는 특성 공간의 각 영역에서 다른 구조를 포착하기 어렵다.

### 6.2 방법

입력 공간을 여러 클러스터로 분할하고, 각 클러스터에 대해 **지역 마할라노비스 거리**를 학습한다:

$$d_{M_c}(x_i, x_j) = \sqrt{(x_i - x_j)^T M_c (x_i - x_j)}$$

여기서 c는 클러스터 인덱스이다.

## 7. 논문에서 얻는 통찰

1. **거리의 선택은 알고리즘의 선택만큼 중요하다**: KNN의 성능은 알고리즘 자체보다 거리 메트릭에 더 크게 의존할 수 있다. 적절한 거리 메트릭을 학습하면 단순한 KNN이 복잡한 SVM을 능가할 수 있다.

2. **마진의 보편적 유용성**: SVM에서 성공적이었던 대규모 마진 원리가 KNN에서도 효과적이다. 이는 마진 극대화가 분류 문제에서 보편적으로 유용한 원리임을 시사한다.

3. **볼록 최적화의 실용적 가치**: LMNN이 볼록 문제로 정식화되어 전역 최적해를 보장한다는 것은 실용적으로 매우 중요하다. 비볼록 문제에서 흔한 지역 최적(local optima) 문제를 피할 수 있다.

4. **감독 차원 축소**: LMNN은 분류 목적에 최적화된 차원 축소를 자연스럽게 제공한다. 이는 PCA(비감독) + KNN보다 우수한 이유를 설명한다.

5. **딥러닝과의 연결**: LMNN의 아이디어는 딥러닝에서의 메트릭 학습(triplet loss, contrastive learning)으로 자연스럽게 확장된다. 선형 변환 L을 신경망으로 대체하면 비선형 거리 학습이 가능하다.
