### 1.2 통계적 학습의 기본 구조

#### 한국어 해설

모든 지도학습 문제의 공통 구조:

1. **학습 데이터**: $\mathcal{T} = \{(x_1, y_1), (x_2, y_2), \ldots, (x_N, y_N)\}$
2. **목표**: 새로운 입력 $x_{\text{new}}$에 대해 $y_{\text{new}}$를 잘 예측하는 함수 $\hat{f}$ 학습
3. **손실 함수**: 예측의 좋고 나쁨을 정량화
   - 회귀: $L(y, \hat{y}) = (y - \hat{y})^2$ (제곱 오차)
   - 분류: $L(y, \hat{y}) = I(y \neq \hat{y})$ (0-1 손실)

## Chapter 2: Overview of Supervised Learning (지도학습 개요)

### 2.1 Two Simple Approaches: Least Squares and k-NN

#### 최소제곱법 (Least Squares) 상세 해설

**모델:**

$$\hat{Y} = X^T \hat{\beta}$$

여기서:
- $X = (X_1, X_2, \ldots, X_p)^T$: $p$차원 입력 벡터
- $\hat{\beta} = (\hat{\beta}_0, \hat{\beta}_1, \ldots, \hat{\beta}_p)^T$: 회귀 계수 벡터

**최적화 (최소제곱법):**

$$\hat{\beta} = \arg\min_{\beta} \sum_{i=1}^{N} (y_i - x_i^T \beta)^2$$

행렬 형태로:

$$\hat{\beta} = \arg\min_{\beta} (y - X\beta)^T(y - X\beta)$$

**해석적 해 (Closed-form Solution):**

$$\hat{\beta} = (X^T X)^{-1} X^T y$$

**분류에의 적용:**
- K개 클래스 각각에 대해 지시 변수(indicator variable) 생성
- 각 클래스에 대해 선형 회귀 수행
- 예측: $\hat{G}(x) = \arg\max_k \hat{f}_k(x)$

**한계:**
- 선형 가정이 위반되면 큰 편향
- 결정 경계가 선형으로 제한됨
- 고차원에서 다중공선성 문제

#### k-최근접 이웃법 (k-Nearest Neighbors) 상세 해설

**모델:**

$$\hat{Y}(x) = \frac{1}{k} \sum_{x_i \in N_k(x)} y_i$$

여기서 $N_k(x)$는 $x$에 가장 가까운 $k$개의 학습 데이터 포인트

**분류의 경우:**

$$\hat{G}(x) = \arg\max_j \frac{1}{k} \sum_{x_i \in N_k(x)} I(y_i = j)$$

(다수결 투표)

**k의 역할:**
- $k = 1$: 가장 가까운 이웃 하나만 사용 → 매우 유연 → 높은 분산
- $k = N$: 모든 데이터의 평균 → 매우 부드러움 → 높은 편향
- $k$가 증가할수록: 편향 증가, 분산 감소

#### 두 방법의 대비

```
    최소제곱법                     k-NN (k=1)
    결정 경계: 직선               결정 경계: 복잡한 곡선

    │ + + +  /  - - -            │ +  /─\  -  /─\
    │ + + / - - - -              │ + │   │- -│   │-
    │ + / - - - - -              │ +  \_/ -   \_/ -
    │ / - - - - - -              │ + + - - - + + - -
    └─────────────               └─────────────────

    편향: 높음, 분산: 낮음          편향: 낮음, 분산: 높음
    안정적 결정 경계               불안정한 결정 경계
```

### 2.2 Statistical Decision Theory (통계적 결정 이론)

#### 한국어 해설

이 섹션은 Chapter 2의 이론적 핵심이다.

**기대 예측 오류 (Expected Prediction Error, EPE):**

$$EPE(f) = E_{X,Y}[L(Y, f(X))] = \int\int L(y, f(x)) \, p(x, y) \, dx \, dy$$

**제곱 오차 손실의 경우:**

$$EPE(f) = E[(Y - f(X))^2]$$

조건부 기대값으로 분해:

$$EPE(f) = E_X\left[E_{Y|X}[(Y - f(X))^2 | X]\right]$$

각 $x$에서 독립적으로 최소화하면:

$$f^*(x) = E[Y | X = x] \quad \text{(조건부 평균)}$$

이것이 **회귀 함수(regression function)**이며, 제곱 오차 손실 하에서의 최적 예측기이다.

**0-1 손실의 경우 (분류):**

$$EPE(G) = E[I(Y \neq G(X))] = 1 - E[\max_k P(G = k | X)]$$

최적 분류기:

$$G^*(x) = \arg\max_k P(G = k | X = x) \quad \text{(베이즈 분류기)}$$

**핵심 메시지:** 최적 예측기는 조건부 분포 $P(Y|X)$에 의해 완전히 결정된다. 그러나 $P(Y|X)$를 모르므로, 학습 데이터로부터 이를 추정해야 한다.

### 2.3 The Bias-Variance Decomposition (편향-분산 분해)

#### 한국어 해설

이 섹션은 1장의 "편향-분산 트레이드오프"의 수학적 기반이다.

**EPE의 분해:**

학습 데이터 $\mathcal{T}$에 대한 기대값을 취하면, 특정 지점 $x_0$에서의 EPE는:

$$EPE(x_0) = E_{\mathcal{T}}[(Y - \hat{f}(x_0))^2 | X = x_0]$$

이를 분해하면:

$$EPE(x_0) = \text{Bias}^2(\hat{f}(x_0)) + \text{Var}(\hat{f}(x_0)) + \sigma^2_\epsilon$$

여기서:

**편향 (Bias):**
$$\text{Bias}(\hat{f}(x_0)) = E_{\mathcal{T}}[\hat{f}(x_0)] - f^*(x_0)$$

- 모델 가정으로 인한 체계적 오차
- 선형 모델이 비선형 데이터에 적합할 때 큰 편향

**분산 (Variance):**
$$\text{Var}(\hat{f}(x_0)) = E_{\mathcal{T}}[(\hat{f}(x_0) - E_{\mathcal{T}}[\hat{f}(x_0)])^2]$$

- 학습 데이터의 변동에 의한 예측의 불안정성
- 복잡한 모델(k-NN에서 k=1 등)일 때 큰 분산

**비가약 오류 (Irreducible Error):**
$$\sigma^2_\epsilon = \text{Var}(Y | X = x_0)$$

- 데이터 자체의 노이즈, 어떤 모델로도 줄일 수 없음

### 핵심 수식 요약

$$\boxed{EPE = \text{Bias}^2 + \text{Variance} + \text{Irreducible Noise}}$$

### 그림 설명: 편향-분산 트레이드오프

```
오류 |
     |  \          일반화 오류 (EPE)
     |   \        /
     |    \      /
     |     \    /      분산 (Variance)
     |      \  / ___________/
     |       \/  /
     |        \/
     |        /\
     |  ____/  \
     | /        편향² (Bias²)
     |/________________________  σ² (비가약 오류)
     +---+---+---+---+---+---> 모델 복잡도
     단순          복잡
     (고편향/        (저편향/
      저분산)         고분산)
```

### 2.4 차원의 저주 (The Curse of Dimensionality)

#### 한국어 해설

고차원 공간에서 k-NN의 성능이 급격히 저하되는 현상을 분석한다.

**핵심 결과:**
$p$차원 단위 초구(unit hypercube)에서 전체 데이터의 비율 $r$을 포함하는 이웃의 한 변 길이:

$$e_p(r) = r^{1/p}$$

예시:
- $p = 1$: 10%의 데이터를 포함하려면 구간의 10% 필요
- $p = 10$: 10%의 데이터를 포함하려면 각 차원의 80% 필요 ($0.1^{1/10} \approx 0.794$)
- $p = 100$: 10%의 데이터를 포함하려면 각 차원의 97.7% 필요

**의미:** 고차원에서는 "이웃"이 더 이상 "가까이"에 있지 않다. 국소적(local) 방법이 사실상 전역적(global)이 되어버린다.

### 2.5 Structured Regression Models (구조화된 회귀 모델)

#### 한국어 해설

차원의 저주를 극복하기 위해 **구조적 가정(structural assumptions)**이 필요하다.

**일반적 접근:**
- 부드러움 가정 (smoothness): $f(x)$가 $x$의 작은 변화에 대해 부드럽게 변함
- 가산 모델 (additive model): $f(X) = \sum_{j=1}^{p} f_j(X_j)$
- 선형 모델: $f(X) = X^T \beta$
- 저차원 구조: 효과적 차원이 입력 차원보다 훨씬 낮음

이러한 가정이 적절하면 차원의 저주를 극복할 수 있지만, 잘못된 가정은 큰 편향을 초래한다.

## 핵심 인사이트

### ESL Ch1-2가 주는 핵심 메시지

1. **두 가지 극단의 이해**: 최소제곱법(최소 복잡도)과 k-NN(최대 유연성)은 편향-분산 스펙트럼의 양 끝단. 실무에서 사용하는 대부분의 알고리즘은 이 사이에 위치.

2. **최적 예측기의 존재**: 이론적으로 최적의 예측은 $E[Y|X]$ (회귀) 또는 $\arg\max_k P(G=k|X)$ (분류)이며, 모든 학습 알고리즘은 이를 근사하는 시도.

3. **트레이드오프의 불가피성**: 편향과 분산을 동시에 줄이는 것은 불가능. 데이터와 문제의 특성에 맞는 최적의 균형점을 찾아야 함.

4. **구조의 활용**: 차원의 저주를 극복하려면 문제의 구조(부드러움, 가산성, 희소성 등)를 적극 활용해야 함.

5. **통계학과 ML의 연결**: ESL은 머신러닝을 통계학의 자연스러운 확장으로 바라보며, 두 분야의 연결고리를 명확히 함.
