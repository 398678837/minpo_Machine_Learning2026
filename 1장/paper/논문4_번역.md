## Part 1: 학습 문제의 형식화 (Setting of the Learning Problem)

### 1.1 학습의 일반적 모형

#### 한국어 해설

Vapnik은 지도학습(supervised learning) 문제를 다음과 같이 형식화한다:

**기본 설정:**
- 독립적이고 동일한 분포(i.i.d.)를 따르는 데이터 $(x_1, y_1), (x_2, y_2), \ldots, (x_n, y_n)$이 주어짐
- 각 $(x_i, y_i)$는 알려지지 않은 확률 분포 $P(x, y)$에서 추출
- 목표: 함수 공간 $\mathcal{F}$에서 손실을 최소화하는 함수 $f$를 찾는 것

**실제 위험 (True/Expected Risk):**

$$R(f) = \int L(f(x), y) \, dP(x, y) = E[L(f(x), y)]$$

여기서:
- $L(f(x), y)$: 손실 함수 (예: 제곱 오차, 0-1 손실)
- $P(x, y)$: 데이터의 참 분포 (알려지지 않음)

**문제**: $P(x, y)$를 모르므로 $R(f)$를 직접 계산할 수 없다.

### 1.2 경험적 위험 최소화 (Empirical Risk Minimization, ERM)

#### 한국어 해설

$R(f)$를 직접 계산할 수 없으므로, 학습 데이터에서의 평균 손실로 대체:

**경험적 위험 (Empirical Risk):**

$$R_{\text{emp}}(f) = \frac{1}{n} \sum_{i=1}^{n} L(f(x_i), y_i)$$

**ERM 원리:** $R_{\text{emp}}(f)$를 최소화하는 $f$를 선택

$$\hat{f}_{\text{ERM}} = \arg\min_{f \in \mathcal{F}} R_{\text{emp}}(f)$$

**문제점:** $\mathcal{F}$가 너무 복잡하면 $R_{\text{emp}}$를 0으로 만들 수 있지만 (학습 데이터를 완벽히 맞추지만), $R(f)$는 매우 클 수 있다 → **과적합(Overfitting)**

이것이 바로 "왜 학습 데이터에서의 좋은 성능이 새로운 데이터에서의 좋은 성능을 보장하지 않는가?"에 대한 형식적 설명이다.

## Part 2: VC 이론 (Vapnik-Chervonenkis Theory)

### 2.1 VC 차원의 정의

#### 한국어 해설

**산산조각내기 (Shattering):**
가설 공간 $\mathcal{H}$가 데이터 점 집합 $\{x_1, \ldots, x_h\}$를 "산산조각낸다(shatters)"함은, 이 $h$개의 점에 대한 모든 $2^h$가지 이진 레이블 조합을 $\mathcal{H}$ 안의 어떤 가설로 분류할 수 있다는 뜻이다.

**VC 차원 (VC dimension):**

$$\text{VCdim}(\mathcal{H}) = \max\{h : \mathcal{H} \text{가 크기 } h \text{의 어떤 점 집합을 shatter할 수 있음}\}$$

#### 구체적 예시

**예시 1: 1차원 임계값 분류기** $h(x) = \text{sign}(x - \theta)$
- 1개의 점: 항상 shatter 가능 → VCdim $\geq$ 1
- 2개의 점: $x_1 < x_2$일 때 $(+,-)$과 $(-,+)$은 가능하지만, 다른 조합도 가능해야 shatter
  - $(+,+)$: $\theta < x_1$으로 가능
  - $(-,-)$: $\theta > x_2$로 가능
  - $(+,-)$: 불가능 ($x_1 < x_2$이므로)
- 따라서 VCdim = 1

**예시 2: 2차원 선형 분류기** $h(x) = \text{sign}(w_1 x_1 + w_2 x_2 + b)$
- 3개의 점 (일반 위치): 모든 $2^3 = 8$가지 분류 가능 → VCdim $\geq$ 3
- 4개의 점: 반드시 shatter 불가능한 배치 존재 (XOR 패턴)
- 따라서 VCdim = 3

**일반화: d차원 선형 분류기의 VCdim = d + 1**

### 2.2 VC 일반화 한계 (VC Generalization Bound)

#### 한국어 해설

**정리 (VC Bound):**
확률 $1 - \delta$ 이상으로 다음이 성립:

$$R(f) \leq R_{\text{emp}}(f) + \sqrt{\frac{h(\ln(2n/h) + 1) - \ln(\delta/4)}{n}}$$

여기서:
- $R(f)$: 실제 위험 (일반화 오류)
- $R_{\text{emp}}(f)$: 경험적 위험 (학습 오류)
- $h$: VC 차원
- $n$: 학습 데이터 크기
- $\delta$: 신뢰 수준

#### 직관적 해석

$$\text{일반화 오류} \leq \text{학습 오류} + \text{복잡도 패널티}(\frac{\text{VC 차원}}{\text{데이터 크기}})$$

- **VC 차원이 높을수록**: 복잡도 패널티 증가 → 과적합 위험 증가
- **데이터가 많을수록**: 복잡도 패널티 감소 → 일반화 보장 강화
- **학습 오류가 낮을수록**: 일반화 오류의 상한도 감소

### 핵심 그림: 구조적 위험 최소화

```
오류 |
     |  \                    /
     |   \  일반화 오류     /
     |    \   상한        /
     |     \            /
     |      \__      __/
     |         \    /
     |    학습   \/    복잡도
     |    오류   /\    패널티
     |         /  \
     |        /    \___________
     |       /
     +---+---+---+---+---+---+---> 모델 복잡도 (VC 차원)
         S₁  S₂  S₃  S₄  S₅
              ↑
          최적 복잡도
```

- $S_1 \subset S_2 \subset S_3 \subset \ldots$: 점점 복잡해지는 가설 공간
- 학습 오류: 복잡도 증가 → 감소 (더 많은 함수를 표현 가능)
- 복잡도 패널티: 복잡도 증가 → 증가
- 일반화 오류 상한: 두 항의 합 → U자형 → 최적점이 존재

## Part 3: 서포트 벡터 머신 (Support Vector Machines)

### 3.1 선형 SVM

#### 한국어 해설

**문제 설정:** 이진 분류 ($y_i \in \{-1, +1\}$)에서 최적의 초평면(hyperplane) 찾기

**결정 함수:** $f(x) = \text{sign}(w \cdot x + b)$

**최적화 문제 (하드 마진):**

$$\min_{w, b} \frac{1}{2}\|w\|^2$$

$$\text{subject to } y_i(w \cdot x_i + b) \geq 1, \quad \forall i$$

**마진:** $\gamma = \frac{2}{\|w\|}$

**직관:** $\|w\|$를 최소화 = 마진을 최대화 = VC 차원을 제한 = 일반화 성능 향상

#### 왜 마진 최대화가 좋은가?

Vapnik은 마진이 클수록 VC 차원의 상한이 줄어듦을 증명:

$$\text{VCdim} \leq \min\left(\frac{R^2}{\gamma^2}, d\right) + 1$$

여기서:
- $R$: 데이터를 포함하는 최소 구의 반경
- $\gamma$: 마진
- $d$: 입력 차원

따라서 **마진이 클수록 effective VC 차원이 낮아지고, 일반화 오류의 상한이 줄어든다.**

### 3.2 소프트 마진 SVM

#### 한국어 해설

현실 데이터는 완벽하게 선형 분리가 불가능한 경우가 많다. 이를 위해 **슬랙 변수(slack variables)** $\xi_i$를 도입:

$$\min_{w, b, \xi} \frac{1}{2}\|w\|^2 + C \sum_{i=1}^{n} \xi_i$$

$$\text{subject to } y_i(w \cdot x_i + b) \geq 1 - \xi_i, \quad \xi_i \geq 0$$

- $C$: 정규화 파라미터 (과적합 방지 vs 학습 오류 허용의 트레이드오프)
- $C$가 크면: 학습 오류 최소화 우선 (과적합 위험)
- $C$가 작으면: 마진 최대화 우선 (과소적합 위험)

### 3.3 커널 트릭 (Kernel Trick)

#### 한국어 해설

선형 분리가 불가능한 데이터를 고차원 공간으로 매핑하여 선형 분리 가능하게 만드는 기법.

**핵심 아이디어:**
1. 매핑 함수 $\phi: \mathbb{R}^d \rightarrow \mathbb{R}^D$ ($D \gg d$)
2. 고차원 공간에서 선형 SVM 적용
3. 핵심: $\phi$를 직접 계산하지 않고, 커널 함수만으로 계산 가능

**커널 함수:** $K(x_i, x_j) = \phi(x_i) \cdot \phi(x_j)$

**주요 커널:**

| 커널 | 수식 | 특성 |
|------|------|------|
| 선형 | $K(x, z) = x \cdot z$ | 가장 단순, 선형 분리 가능 시 |
| 다항식 | $K(x, z) = (x \cdot z + c)^d$ | 다항식 결정 경계 |
| RBF (가우시안) | $K(x, z) = \exp(-\gamma\|x-z\|^2)$ | 가장 범용적, 무한 차원 매핑 |
| 시그모이드 | $K(x, z) = \tanh(\kappa x \cdot z + c)$ | 신경망과 유사 |

**커널 트릭의 위력:**
- RBF 커널은 무한 차원 공간으로 매핑하지만, 커널 함수 계산은 $O(d)$
- 따라서 계산 비용 없이 매우 복잡한 결정 경계를 학습 가능

### 3.4 쌍대 문제 (Dual Problem)

#### 한국어 해설

SVM의 최적화는 라그랑주 쌍대 문제로 변환하여 풀 수 있다:

$$\max_{\alpha} \sum_{i=1}^{n} \alpha_i - \frac{1}{2}\sum_{i,j} \alpha_i \alpha_j y_i y_j K(x_i, x_j)$$

$$\text{subject to } 0 \leq \alpha_i \leq C, \quad \sum_{i=1}^{n} \alpha_i y_i = 0$$

- $\alpha_i > 0$인 데이터 포인트가 **서포트 벡터**
- 결정 함수: $f(x) = \text{sign}\left(\sum_{i:\alpha_i > 0} \alpha_i y_i K(x_i, x) + b\right)$
- 서포트 벡터만으로 결정 경계가 결정됨 (데이터의 일부만 중요)

## Part 4: 핵심 인사이트

### Vapnik의 이론이 주는 근본적 메시지

1. **학습의 가능성 조건**: 유한한 VC 차원을 가진 가설 공간에서, 충분한 데이터가 있으면, 경험적 위험이 실제 위험에 수렴한다 (균일 수렴, uniform convergence).

2. **복잡도-일반화 트레이드오프**: 모델이 복잡할수록 학습 오류는 줄지만, 복잡도 패널티가 증가하여 일반화 오류가 커질 수 있다. 최적의 복잡도가 존재한다.

3. **마진의 역할**: SVM에서 마진은 단순한 기하학적 개념이 아니라, VC 차원을 제어하는 정규화 역할을 한다.

4. **데이터 크기의 중요성**: VC 한계에서 복잡도 패널티는 $\sqrt{h/n}$에 비례 → 데이터가 많을수록 더 복잡한 모델을 안전하게 사용 가능.

### 현대적 관점

- Vapnik의 VC 이론은 전통적 ML에는 잘 맞지만, 딥러닝에서는 VC 차원이 데이터 수보다 훨씬 큰데도 잘 일반화되는 현상이 관찰됨 (double descent 현상)
- 이는 VC 이론의 한계가 아니라, 딥러닝의 귀납적 편향(implicit regularization)이 VC 이론이 고려하지 않는 추가 구조를 제공하기 때문
- 그럼에도 Vapnik의 프레임워크는 학습의 본질을 이해하는 가장 기본적인 도구

### 대학원 수준 학습 가이드

- 1단계: VC 차원의 직관적 이해와 간단한 예시 계산
- 2단계: VC 한계의 수학적 유도 이해
- 3단계: SRM 원리의 이해와 정규화와의 연결
- 4단계: SVM의 최적화 문제 유도와 풀이
- 5단계: 커널 트릭의 이론적 기반 (Mercer's theorem)
- 6단계: 현대 딥러닝에서의 일반화 이론과의 비교
