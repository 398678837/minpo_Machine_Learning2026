# 논문 2 요약: A Comparison of Event Models for Naive Bayes Text Classification

## 서지 정보
- **저자**: A. McCallum, K. Nigam
- **제목**: A Comparison of Event Models for Naive Bayes Text Classification
- **학회**: AAAI-98 Workshop on Learning for Text Categorization
- **연도**: 1998
- **다운로드 링크**: https://www.cs.cmu.edu/~knigam/papers/multinomial-aaaiws98.pdf

---

## 핵심 요약

McCallum과 Nigam(1998)의 논문은 나이브 베이즈 텍스트 분류에서 사용되는 두 가지 **이벤트 모델(event model)**을 체계적으로 비교한 중요한 연구이다. 텍스트를 확률적으로 모델링하는 방식에 따라 나이브 베이즈의 성능이 크게 달라질 수 있음을 보이며, **다항 이벤트 모델(Multinomial Event Model)**이 **다변량 베르누이 이벤트 모델(Multi-variate Bernoulli Event Model)**보다 텍스트 분류에 더 적합함을 실험적으로 증명한다.

### 핵심 기여 (Key Contributions)

1. **두 가지 이벤트 모델의 명확한 구분**: 텍스트의 나이브 베이즈 분류에서 흔히 혼동되는 두 가지 확률 모델을 명확히 구분한다:

   **(a) 다변량 베르누이 모델 (Multi-variate Bernoulli Model = BernoulliNB)**
   - 각 단어의 **존재/부재(presence/absence)**만 고려 (이진 값)
   - 문서는 어휘 사전 크기의 이진 벡터로 표현
   - P(문서 | 클래스) = prod_w P(w 존재 | 클래스)^{x_w} * P(w 부재 | 클래스)^{1-x_w}
   - 여기서 x_w는 단어 w의 존재 여부 (0 또는 1)

   **(b) 다항 모델 (Multinomial Model = MultinomialNB)**
   - 각 단어의 **출현 빈도(frequency)**를 고려
   - 문서는 단어 빈도 벡터로 표현
   - P(문서 | 클래스) = (n! / prod_w n_w!) * prod_w P(w | 클래스)^{n_w}
   - 여기서 n_w는 단어 w의 출현 횟수

2. **다항 모델의 우월성 증명**: 다양한 텍스트 분류 벤치마크에서 다항 모델이 베르누이 모델보다 일관되게 우수한 성능을 보임을 실험적으로 증명한다. 핵심 이유:

   - 단어의 빈도 정보가 분류에 중요한 신호를 제공한다
   - "machine"이 10번 등장하는 문서는 1번 등장하는 문서보다 기계학습 관련 문서일 가능성이 높다
   - 베르누이 모델은 이러한 빈도 정보를 버린다

3. **어휘 사전 크기의 영향 분석**: 어휘 사전의 크기가 두 모델에 미치는 영향이 다름을 보인다:
   - **베르누이 모델**: 어휘 사전이 작을 때 상대적으로 양호 (소규모 어휘에서의 존재/부재가 의미 있음)
   - **다항 모델**: 어휘 사전이 클 때 더 큰 이점 (풍부한 빈도 정보 활용)

4. **문서 길이의 영향 분석**: 긴 문서에서 다항 모델의 이점이 더 커짐을 보인다. 긴 문서에서는 단어 빈도의 차이가 더 뚜렷해져, 빈도 정보가 더 유용하기 때문이다.

5. **라플라스 스무딩의 중요성**: 두 모델 모두에서 라플라스 스무딩(alpha=1)의 효과를 분석하며, 특히 다항 모델에서 스무딩의 영향이 크다는 것을 보인다.

### 논문의 영향

이 논문은 텍스트 분류를 위한 나이브 베이즈의 **표준 참고 문헌**으로, sklearn에서 MultinomialNB와 BernoulliNB를 별도 클래스로 제공하는 직접적 근거가 되었다. 이후 텍스트 분류에서 MultinomialNB가 기본 선택(default choice)으로 자리잡는 데 결정적 역할을 하였다.

### 한계점

- 이진 분류 위주의 실험으로, 다중 클래스에서의 비교가 부족하다.
- TF-IDF 가중치와의 조합 효과를 분석하지 않는다.
- 라플라스 스무딩 이외의 스무딩 기법과의 비교가 없다.
