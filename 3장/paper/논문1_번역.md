## 1. 서론: 파이썬 통계 컴퓨팅의 간극

2010년 당시, 파이썬은 과학 컴퓨팅에서 이미 상당한 위상을 가지고 있었지만, **통계 분석과 데이터 조작** 측면에서는 R에 비해 뚜렷한 한계가 있었다.

### 기존 도구의 한계
| 도구 | 한계 |

## 2. 핵심 데이터 구조

### 2.1 Series

Series는 **인덱스(라벨)가 부여된 1차원 배열**이다.

```python
import pandas as pd

# Series 생성
s = pd.Series([4, 7, -5, 3], index=['d', 'b', 'a', 'c'])
print(s)
# d    4
# b    7
# a   -5
# c    3
# dtype: int64

# 라벨로 접근
print(s['b'])  # 7

# 벡터화 연산
print(s * 2)
# d     8
# b    14
# a   -10
# c     6
```

내부적으로 Series는 NumPy ndarray를 감싸고(wrapping) 있으며, 여기에 인덱스 레이어를 추가한 것이다.

### 2.2 DataFrame

DataFrame은 **각 열이 서로 다른 타입을 가질 수 있는 2차원 표 구조**이다.

```python
data = {
    '이름': ['홍길동', '김철수', '이영희'],
    '나이': [25, 30, 28],
    '점수': [85.5, 92.3, 88.1]
}
df = pd.DataFrame(data)
#      이름  나이   점수
# 0  홍길동   25  85.5
# 1  김철수   30  92.3
# 2  이영희   28  88.1
```

DataFrame의 각 열은 Series이며, 열마다 독립적인 dtype을 가진다:
- '이름' 열: object (문자열)
- '나이' 열: int64
- '점수' 열: float64

## 3. 자동 데이터 정렬 (Automatic Data Alignment)

Pandas의 가장 혁신적인 기능 중 하나이다. 서로 다른 인덱스를 가진 Series/DataFrame 간 연산 시, 인덱스를 자동으로 맞춰준다.

```python
s1 = pd.Series([1, 2, 3], index=['a', 'b', 'c'])
s2 = pd.Series([4, 5, 6], index=['b', 'c', 'd'])

result = s1 + s2
# a    NaN   # s2에 'a' 없음 -> NaN
# b    6.0   # 2 + 4
# c    8.0   # 3 + 5
# d    NaN   # s1에 'd' 없음 -> NaN
```

이 기능은 불완전한 데이터셋 간의 연산에서 발생할 수 있는 오류를 방지하고, 코드의 안전성을 크게 높인다.

## 4. 결측치 처리 (Missing Data Handling)

### 4.1 결측치 표현
Pandas는 `NaN`(Not a Number)을 결측치의 표준 표현으로 사용한다.

### 4.2 결측치 처리 함수

| 함수 | 기능 |
|------|------|
| `isna()` / `isnull()` | 결측 여부 확인 (불리언 반환) |
| `notna()` / `notnull()` | 비결측 여부 확인 |
| `fillna(value)` | 결측치를 특정 값으로 대체 |
| `dropna()` | 결측치가 있는 행/열 삭제 |
| `interpolate()` | 보간법으로 결측치 채우기 |

```python
df = pd.DataFrame({'A': [1, np.nan, 3], 'B': [4, 5, np.nan]})

# 결측치 확인
print(df.isna().sum())  # A: 1, B: 1

# 평균으로 채우기
df_filled = df.fillna(df.mean())

# 결측치 있는 행 삭제
df_dropped = df.dropna()
```

## 5. GroupBy: 분할-적용-결합 패턴

McKinney는 Hadley Wickham의 "split-apply-combine" 전략을 Pandas에 구현했다.

### 작동 과정
```
원본 데이터 --[Split]--> 그룹1, 그룹2, ...
                         --[Apply]--> 집계1, 집계2, ...
                                      --[Combine]--> 결과 DataFrame
```

```python
# 카테고리별 평균
df.groupby('카테고리')['매출'].mean()

# 다중 집계
df.groupby('카테고리').agg({
    '매출': ['sum', 'mean', 'count'],
    '수량': ['sum', 'max']
})

# 변환 (그룹별 표준화)
df['매출_표준화'] = df.groupby('카테고리')['매출'].transform(
    lambda x: (x - x.mean()) / x.std()
)
```

## 6. 데이터 결합 (Merge/Join)

SQL의 JOIN 연산에 대응하는 기능을 제공한다.

### merge의 4가지 방식

```python
# Inner Join: 양쪽 모두에 있는 키만
pd.merge(left, right, on='key', how='inner')

# Outer Join: 모든 키 포함
pd.merge(left, right, on='key', how='outer')

# Left Join: 왼쪽 테이블의 모든 키 유지
pd.merge(left, right, on='key', how='left')

# Right Join: 오른쪽 테이블의 모든 키 유지
pd.merge(left, right, on='key', how='right')
```

## 7. 시계열 데이터 지원

McKinney의 금융 분석 배경이 반영된 강력한 시계열 기능:

```python
# 날짜 인덱스 생성
dates = pd.date_range('2024-01-01', periods=365, freq='D')
ts = pd.Series(np.random.randn(365), index=dates)

# 리샘플링 (일별 -> 월별)
monthly = ts.resample('M').mean()

# 이동 평균
rolling_mean = ts.rolling(window=7).mean()

# 날짜 기반 슬라이싱
ts['2024-03']  # 2024년 3월 데이터만 추출
```
