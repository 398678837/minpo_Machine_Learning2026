## 개요 (300+ words)

본 문서는 Pandas 라이브러리의 공식 소프트웨어 인용 자료로, Pandas의 전체적인 기능과 역량을 포괄적으로 개관하는 참고 자료이다. Zenodo에 등록되어 학술 논문에서 Pandas를 인용할 때 참조하는 공식 DOI를 제공한다.

Pandas는 2008년 Wes McKinney가 AQR Capital Management에서 금융 데이터 분석을 위해 개발을 시작한 이후, 파이썬 데이터 분석의 사실상 표준 라이브러리로 성장하였다. 2020년 기준 월간 다운로드 수가 수백만 건에 달하며, 데이터 과학, 기계학습, 금융, 생명과학, 사회과학 등 거의 모든 데이터 중심 분야에서 사용되고 있다.

Pandas의 핵심 데이터 구조는 Series(1차원)와 DataFrame(2차원)이다. DataFrame은 관계형 데이터베이스의 테이블이나 엑셀 스프레드시트에 대응하는 구조로, 각 열이 서로 다른 데이터 타입(숫자, 문자열, 날짜, 범주형 등)을 가질 수 있다. 내부적으로 NumPy 배열을 기반으로 구현되어 높은 성능을 제공한다.

주요 기능으로는 다음이 있다. 데이터 입출력(CSV, Excel, SQL, JSON, Parquet 등 다양한 형식), 인덱싱과 선택(loc, iloc, 불리언 인덱싱), 데이터 정제(결측치 처리, 중복 제거, 타입 변환), 데이터 변환(apply, map, transform), 그룹별 연산(groupby + agg/transform/filter), 시계열 처리(DatetimeIndex, resample, rolling), 데이터 결합(merge, join, concat), 피벗 테이블(pivot_table, crosstab), 범주형 데이터(Categorical dtype), 문자열 처리(str accessor) 등이 포함된다.

최근 버전에서는 성능 개선을 위한 노력이 지속되고 있다. Apache Arrow 기반의 문자열 타입 도입, Copy-on-Write 최적화, nullable 정수/불리언 타입 등이 추가되었다. 또한 pandas 2.0에서는 PyArrow 백엔드를 선택적으로 사용할 수 있게 되어, 문자열 처리와 결측치 처리의 성능이 크게 향상되었다.

Pandas는 오픈소스 프로젝트로, 수천 명의 기여자가 참여하고 있으며, NumFOCUS의 후원을 받고 있다. 활발한 커뮤니티와 풍부한 문서화, 수많은 서드파티 확장 라이브러리(geopandas, modin, vaex 등)가 Pandas 생태계를 더욱 풍성하게 만들고 있다.

## 핵심 기여 (Key Contributions)

1. **표 형식 데이터의 표준 도구 확립**: DataFrame이 파이썬 데이터 분석의 표준 데이터 구조로 자리잡음
2. **포괄적 데이터 입출력**: CSV, Excel, SQL, JSON, Parquet 등 거의 모든 데이터 형식 지원
3. **강력한 데이터 정제 기능**: 결측치, 중복, 이상치 처리를 위한 체계적 도구 제공
4. **유연한 그룹별 연산**: split-apply-combine 패턴의 직관적 구현
5. **시계열 데이터 전문 기능**: 날짜/시간 인덱싱, 리샘플링, 이동 윈도우 등

## 기계학습 과목과의 관련성

Pandas는 ML 워크플로우의 첫 번째 단계인 데이터 로딩과 전처리에서 핵심적으로 사용된다. 3장에서 학습하는 CSV 파일 로딩, DataFrame 인덱싱, groupby 집계, 데이터 결합은 모든 ML 프로젝트에서 반복되는 기본 작업이다. 이후 장에서 다양한 ML 알고리즘을 적용할 때, 데이터는 항상 pd.read_csv()로 시작하여 전처리 후 NumPy 배열로 변환되어 모델에 입력된다. Pandas는 이 전체 파이프라인에서 빠질 수 없는 핵심 도구이다.
